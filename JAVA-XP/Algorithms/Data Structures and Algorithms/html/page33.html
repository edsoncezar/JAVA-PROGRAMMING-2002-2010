Data Structures and Algorithms with Object-Oriented Design Patterns in Java

Abstract Data Types and the Class Hierarchy

Chapter  introduces the notion of an abstract data type. All of the data structures discussed in this book are presented as instances of various abstract data types. Chapter  also introduces the class hierarchy as well as the various related concepts such as enumerations and visitors.
