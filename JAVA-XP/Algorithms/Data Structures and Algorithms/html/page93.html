Array Indexing Methods-get and put

The elements of a multi-dimensional array are indexed using the get and put methods. For example, you can access the element of a three-dimensional array a like this: 

a.get (new int[] {i, j, k});
and you can modify the element like this: 
a.put (new int[] {i, j, k}, value);
