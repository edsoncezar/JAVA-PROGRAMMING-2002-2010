Binary Search Trees

The class BinarySearchTree introduced in Program  represents binary search trees. Since binary trees and binary search trees are topologically similar, the BinarySearchTree class extends the BinaryTree introduced in Program . In addition, because it represents search trees, the BinarySearchTree class implements the SearchTree interface defined in Program .
