Hash Tables

A hash table  is a searchable container. As such, its interface provides methods for putting an object into the container, finding an object in the container, and removing an object from the container. Program  defines the HashTable interface. The HashTable interface extends the SearchableContainerInterface defined in Program . One additional method, called getLoadFactor, is declared. The purpose of this method is explained in Section .
