Olá,

  Antes de iniciar a execução dos exemplos deste mini-curso proceda os seguintes passos:

0) Faça o download das versões do J2SDK, ANT e ASPECTJ utilizadas na confecção deste mini-curso:
   J2SDK 1.4.2_01 --> download em http:\\java.sun.com
   ANT 1.5.4 --> download em http:\\ant.apache.org
   ASPECTJ 1.1 --> download em http:\\eclipse.org\aspectj

   Proceda a instalação destas ferramentas. 
   A instalação do ASPECTJ envolve somente a execução do comando: "java -jar aspectj-1.1.0.jar" e a escolha do diretório de instalação.
   A instalação do ANT envolve a somente a descompactação do arquivo em algum diretório.

1) Edite o arquivo java-env.bat e configure as seguintes variáveis de ambiente:
   JAVA_HOME: deve apontar para a diretório de instalação do J2SDK
   ANT_HOME: deve apontar para o diretório de instalação do ANT
   ASPECTJ_HOME: deve apontar para o diretório de instalação do ASPECTJ

2) Inicie um novo shell através da chamada de um dos arquivos abaixo:
   java-env-win9x.bat --> se seu sistema operacional é Windows 95 ou 98
   java-env-winnt.bat --> se seu sistema operacional é Windows NT, 2000 ou XP
   (Sugestão: crie um link para um destes arquivos no seu desktop)

3) Os exemplos a serem seguidos estão abaixo do diretório src. Para visualizar a execução dos programas de teste basta invocar o ANT. Exemplo:

   E:\mini-curso-aop>cd src\01-Mensageiro\2-olaAspecto
   E:\mini-curso-aop>ant
