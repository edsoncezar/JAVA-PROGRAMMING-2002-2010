GLOSSÁRIO
J2EE 2 Platform, Enterprise Edition

Este glossário define termos de acordo com seu uso no contexto da JAVA 2 Platform, Enterpresie Edition (Plataforma J2EE). Muitos dos termos possue uso mais genérico. Por exemplo, o termo handle é muito usado nas linguagens de programação modernas, mas aqui ele é definido em termos do seu uso na tecnologia Enterprise JavaBeans(EJB)

A B C D E F G H I J L M N O P Q R S T U V X Z

XML
Extensible Markup Language. Uma linguagem de marcação que lhe permite definir as tags (marcação) necessárias para identificar o conteúdo, dados e texto, em documentos XML. Ela é diferente da HTML, a linguagem de marcação mais usada para apresentar informações na Internet. HTML possui tags fixas, que lidam principalmente com estilo ou apresentação. Um documento XML precisa sofrer uma transformação para uma linguagem com tags de estilo sob o controle de uma folha de estilo antes que possa ser apresentado por um browser ou outro mecanismo de apresentação. Dois tipos de folhas de estilo usadas com XML são CSS e XSL. Veja "CSS" e "XSLT", neste glossário. Normalmente, a XML é transformada em HTML para apresentação. Embora as tags possam ser definidas como for preciso na criação de um documento XML, uma Document Type Definition (DTD) pode ser usada para se definir os elementos permitidos em um tipo de documento qualquer. Veja "DTD", neste glosário. Um documento pode ser comparado com as regras na DTD para determinar sua validade e para alocar elementos em particular no documento. Os descritores de distribuição J2EE são expressos em XML, com DTDs definindo os elementos permitidos. Os programas para processar documentos XML normalmente utilizam APIs SAX ou DOM. Veja "SAX" e "DOM", neste glossário.

XSL
Extensible Stylesheet Language. Uma linguagem de transformação XML usada para transformar documentos XML em documentos com tags de objeto de fluxo, para fins de apresentação. O aspecto de transformação da XLS foi separado para a XSLT, com o nome XSL sendo usado agora para designar a linguagem de fluxo de apresentação. XSL é uma descendente direta da linguagem de estilo DSSSL para a SGML (Standard Generalized Markup Language), a linguagem da qual a XML foi extraída. Ela foi designada para ter todas as capacidades da CSS, a folha de estilo normalmente utilizada com HTML. Os objetos de fluxo da XSL podem ser apresentados por browsers especializados e eles mesmos tansformados em documentos PDF para apresentação pelo produto Acrobat, da Adobe. Veja "HTML", "XML", "CSS" e "XSLT", neste glossário.

XSLT
XSL Transformation. Um arquivo XML que contola a transformação de um documento XML para outro documento XML ou para HTML. O documento de destino normalmente terá tags relacionadas à apresentação, indicando como ele será exibido por um browser ou por outro mecanismo de apresentação. XSLT inicialmente fazia parte da XSL que também incluía uma linguagem de tag de objetos de fluxo de estilo.
