GLOSSÁRIO
J2EE 2 Platform, Enterprise Edition

Este glossário define termos de acordo com seu uso no contexto da JAVA 2 Platform, Enterpresie Edition (Plataforma J2EE). Muitos dos termos possue uso mais genérico. Por exemplo, o termo handle é muito usado nas linguagens de programação modernas, mas aqui ele é definido em termos do seu uso na tecnologia Enterprise JavaBeans(EJB)

A B C D E F G H I J L M N O P Q R S T U V X Z

IDL
Interface Definition Language. Uma linguagem usada para se definir interfaces com objetos CORBA remotos. As definições de interface são independntes dos sistemas operacionais e das linguagens de programação. Um compilador IDL2Java é usado para obter uma representação Java de interfaces IDL. Existem compiladores similares para outra linguagens de computador e sistemas de programação. Veja também CORBA, neste glossário.

IIOP
Internet Inter-ORB Protocol. Um protocolo usado para a comunicação entre agentes de pedido de objeto CORBA(ORBs). Veja também ORB, neste glossário.

interface local
Uma das duas interfaces para um enterprise bean. A interface local(ou home) define zero ou mais métodos para gerenciar um enterprise bean. A interface local em um bean de sessão define métodos create, finder e remove.

interface remota
Uma das duas interfaces para um enterprise bean. A interfrace remota deine os métodos comerciais que podem ser chamados por um cliente.

ISV
Independent Software Vendor.
