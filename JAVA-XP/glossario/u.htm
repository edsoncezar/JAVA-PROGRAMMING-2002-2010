GLOSSÁRIO
J2EE 2 Platform, Enterprise Edition

Este glossário define termos de acordo com seu uso no contexto da JAVA 2 Platform, Enterpresie Edition (Plataforma J2EE). Muitos dos termos possue uso mais genérico. Por exemplo, o termo handle é muito usado nas linguagens de programação modernas, mas aqui ele é definido em termos do seu uso na tecnologia Enterprise JavaBeans(EJB)

A B C D E F G H I J L M N O P Q R S T U V X Z

URI
Uniform Resource Identifier. Uma string de caracteres compacta para identificar um recurso abstrato ou físico. Um URI é um URL ou um URN. URLs e URNs são entidades concretas que realmente existem; um URI é uma superclasse abstrata.

URL
Uniform Resource Locator. Um padrão para a escrita de referência textual a um trecho de dados arbitrário na World Wide Web. Um URL se parece com protocolo://host/infolocal, onde protocolo especifica um protocolo para se apanhar o objeto (como HTTP ou FTP), host especifica o nome da Internet para o host destinado e infolocal é uma string (normalmente um nome de arquivo) passada para o manipulador de protocolo no host remoto.

URN
Uniform Resource Name. Um identificador exclusivo que identifica uma entidade sem especificar onde ela está localizada. Um sistema pode usar um URN para pesquisar uma entidade localmente antes de tentar encontrá-la na Web. O uso de URNs permite que o local da Web mude enquanto ainda permite que a entidade seja encontrada.
