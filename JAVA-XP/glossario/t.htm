GLOSSÁRIO
J2EE 2 Platform, Enterprise Edition

Este glossário define termos de acordo com seu uso no contexto da JAVA 2 Platform, Enterpresie Edition (Plataforma J2EE). Muitos dos termos possue uso mais genérico. Por exemplo, o termo handle é muito usado nas linguagens de programação modernas, mas aqui ele é definido em termos do seu uso na tecnologia Enterprise JavaBeans(EJB)

A B C D E F G H I J L M N O P Q R S T U V X Z

tag JSP
Um pedaço de texto entre um sinal de menor e um sinal de maior, que é usado em um arquivo JSP como parte de um elemento JSP. A tag é considerada marcação, ao contrário de dados, pois é cercada por sinais de menor e maior.

tópico
Veja "sistema de mensagem publicar/assinar", neste glossário.

transação
Uma unidade indivisível de trabalho que modifica dados. Uma transação inclui uma ou mais instruções de programa, todas elas sendo completadas ou canceladas. As transações permitem que vários usuários acessem os mesmos dados de forma simultãnea.

transação controlada pelo conector
Uma transação cujos limites são definidos por um contêiner EJB. Um bean de entidade precisa usar transações controladas por contêiner.

transação controlada por bean
Uma transação cujos limites são definidos por um enterprise bean.
