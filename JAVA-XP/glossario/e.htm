GLOSSÁRIO
J2EE 2 Platform, Enterprise Edition

Este glossário define termos de acordo com seu uso no contexto da JAVA 2 Platform, Enterpresie Edition (Plataforma J2EE). Muitos dos termos possue uso mais genérico. Por exemplo, o termo handle é muito usado nas linguagens de programação modernas, mas aqui ele é definido em termos do seu uso na tecnologia Enterprise JavaBeans(EJB)

A B C D E F G H I J L M N O P Q R S T U V X Z

EJB Contêiner Provider
Um fornecedor que oferece um contêiner EJB.

EJB Server Provider
Um fornecedor que oferece um servidor EJB.

EJB
Veja "Enterprise JavaBeans(EJB)", neste glossário.

elemento de scripting JSP
Uma declaração JSP, scriptlet ou expressão, cuja sintaxe de tag é definida pela especificação JSP, e cujo contéudo é escrito de acordo com a linguagem de scripting usada na página JSP. A especificação JSP descreve a sintaxe e a semântica para o caso em que o atributo de página da linguagem é "java".

elemento JSP
Uma parte de uma página JSP é reconhecida por um tradutor JSP. Um elemento pode ser uma diretiva, uma ação ou um elemento de scripting.

enterprise bean
Um componente que implementa uma tarefa comercial ou uma entidade comercial e reside em um contêiner EJB; um bean de entidade, um bean de sessão ou um bean controlado por mensagem.

Enterprise Bean Provider
Um programador de aplicação que produz classes de enterprise bean, interfaces remotas e domésticas e arquivos de descritor de distribuição, empacotando-os em um arquivo JAR EJB.

Enterprise JavaBeans (EJB)
Uma arquitetura de componente para o desenvolvimento e distribuição de aplicações orientadas a objeto, distribuídas e em âmbito de empresa. As aplicações escritas usando-se a arquitetura Enterprise JavaBeans são escaláveis , transacionais e seguras.

estado de conversação
Os valores de campo de um bean de sessão mais o fechamento transitivo dos objetos alcançados a partir dos campos do bean. O fechamento transitivo de um bean é definido em termos do protocolo de serialização para a linguagem de programação Java, ou seja, os campos que seriam armazenados pela serialização da instância do bean.

expressão JSP
Um elemento de scripting que contém uma expressão válida na linguagem de scripting que é avaliada, convertida em uma string e colocada em um objeto implícito.
