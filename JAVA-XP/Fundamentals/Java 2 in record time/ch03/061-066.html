While we’re working on text fields and buttons, let’s take a look at the multi-line text field called a text area. Java uses a text area to support text that takes up more than one line.

Handling Java Text Areas

A text area really works in almost the same way that a text field does, but it can have several lines, as shown in Figure 3.8.


FIGURE 3.8  An empty text area

This is the control to use when you have multiple lines of text to display, such as a set of instructions, or when you let the user edit a large amount of text. Text fields can do the same job, but for large amounts of text—especially text that has carriage returns or paragraphs—text areas are the way to go.

Creating txtarea.java
