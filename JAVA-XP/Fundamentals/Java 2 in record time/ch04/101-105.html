Your check box example is ready to go. Build the new applet now and run it, as shown below. As you can see, when the user clicks a check box, your applet responds and indicates what happened. Your check box example works exactly as you want it to. The code for this applet appears in checker.java.

TIP:  To see at any time whether a check box is clicked or not, you can use the getState() method like this: checkbox1.getState(), which returns true if the check box is checked and false otherwise.

 import java.applet.Applet;
 import java.awt.*;
 import java.awt.event.*;

 public class checker extends Applet implements ItemListener {

     Checkbox checkbox1, checkbox2, checkbox3, checkbox4, checkbox5;
     TextField text1;

     public void init(){

         checkbox1 = new Checkbox("1");
         add(checkbox1);
         checkbox1.addItemListener(this);
         checkbox2 = new Checkbox("2");
         add(checkbox2);
         checkbox2.addItemListener(this);
         checkbox3 = new Checkbox("3");

         checkbox4 = new Checkbox("4");

         checkbox5 = new Checkbox("5");
         add(checkbox5);
         checkbox5.addItemListener(this);

     }

     public void itemStateChanged(ItemEvent e) {

     }
 }
