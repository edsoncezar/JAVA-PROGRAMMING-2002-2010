Now your new check boxes are installed. The next step is to connect them to the code. You might expect to use an ActionListener interface, but in fact, you use a different interface with check boxes because unlike buttons, check boxes can be either checked or unchecked (and the way to test their state is with the getState() method). The interface you use here is called ItemListener, and you implement that as follows:
