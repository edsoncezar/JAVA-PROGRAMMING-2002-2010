Now you have a menubar with a File menu in it—it’s time to add menu items to this menu:

Each of these items is a separate object of the Java MenuItem class. In this case, you want to add three items to the File menu, so declare three new MenuItem objects, menuitem1, menuitem2, and menuitem3:

 class MenuFrame extends Frame implements ActionListener {

      Menu Menu1;
      MenuBar Menubar1;
→     MenuItem menuitem1, menuitem2, menuitem3;

      TextField text1;

      MenuFrame(String title){
           super(title);
           text1 = new TextField(" ");
           setLayout(new GridLayout(1, 1));
           add(text1);
           Menubar1 = new MenuBar();

           Menu1 = new Menu("File");

      }
 }
