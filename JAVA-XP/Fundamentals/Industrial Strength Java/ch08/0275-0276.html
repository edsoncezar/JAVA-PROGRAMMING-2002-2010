Summary

This chapter taught you how to build your own data structures from scratch and a little bit about application building along the way.

As you go on to explore other data structures, remember that writing a dynamic data structure in a pointer-free language can be a bit confusing. If you are used to managing memory for yourself, you might feel a touch constrained by the fact that Java does not permitted this. As this chapter has shown, however, data structures implemented in Java are actually a touch more elegant than in languages that use pointers. Additionally it is much easier to implement a data structure in Java than it is in a pointer-based language.
