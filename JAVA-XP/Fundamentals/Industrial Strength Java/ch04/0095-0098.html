Applet-Specific Methods

Although an applet is little more than a different form of your Java code, it does contain methods that only an applet can access. These methods basically enable the applet to communicate with the host browser.

Retrieving Parameters with Applet-Specific Methods

The first applet-specific method you will want to study is the getParameter() method contained in class java.applet.Applet. This method enables you to fetch the value attribute of a <PARAM> tag of the HTML page which hosts the applet. The Java code for the applet pictured in figure 4.1 is shown in listing 4.2b. Note specifically the section of the applet singled out in listing 4.2a where parameters are read in from the HTML page.

Listing 4.2aThe getParameter() Method Allows Your Applet to Read In Data from an HTML Page 

myString = this.getParameter("text");
myFont =  new Font("Helvetica", Font.PLAIN, 24);
red = Integer.parseInt(this.getParameter("red"));
green = Integer.parseInt(this.getParameter("green"));
blue = Integer.parseInt(this.getParameter("blue"));
myColor = new Color(red, green, blue);
