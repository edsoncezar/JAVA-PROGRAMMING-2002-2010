WAIT! STOP! Before you send us an e-mail . . . you should be aware that we do not provide technical support at this address! This address is provided so that you can provide us with feedback and let us know of any problems you may be having with the tutorial's content.

Here's where to turn for help with other problems:

  • If you have a Java programming or setup question, hie thee to The Java Developer Connection. It's the best resource we know of, and it's free! You can download early access versions of new software, scan the known bugs in the JDK, search the large database of questions and answers, and much more.
  • If you had trouble downloading or unarchiving the online tutorial, or you're having trouble browsing the tutorial on java.sun.com, please go to the java.sun.com FEEDBACK page and ask the webmaster for help.
  • Save yourself some time and check out our FAQ.
  • Questions about the book? Look at About the Java Tutorial Books and its friends.
