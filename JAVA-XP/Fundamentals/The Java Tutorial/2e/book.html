Errata

If you have comments on The Java Tutorial, or if you've found errors or omissions in the book or CD-ROM, please send mail to tutorial@java.sun.com. If you have general Java programming or setup questions, your best bet is to consult a Java newsgroup such as comp.lang.java.programmer or comp.lang.java.setup, or to sign up with The Java Developer Connection.
