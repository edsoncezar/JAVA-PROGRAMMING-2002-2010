Errata (first edition)

The following is a summary of the errors and ommissions that have been fixed in each printing of the first edition of The Java Tutorial. The links lead to pages that list the errors and ommissions in detail. If you find any that aren't listed here, please send them to tutorial@java.sun.com.

  • The second printing contained ten text corrections in the main text of the book and several corrections to the CD-ROM pages at the end of the book.
  • The third printing is identical to the second printing.
  • The fourth printing contained a lot of text corrections and some content changes. We sent the corrections for this printing to Addison-Wesley on March 18, 1997. Copies of this printing should be out a few weeks after that date. We also re-mastered the CD-ROM for this printing.
