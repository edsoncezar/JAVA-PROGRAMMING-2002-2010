Platform-Specific Details: Viewing an Applet Using the JDK

UNIX: 
appletviewer file:/home/kwalrath/java/Hello.html
DOS shell (Windows 95/NT): 
appletviewer file:/home/kwalrath/java/Hello.html
MacOS:
Start up the AppletViewer. From the File menu, choose OpenURL and enter the URL of the HTML file you created (for example, file:/home/kwalrath/java/Hello.html).
