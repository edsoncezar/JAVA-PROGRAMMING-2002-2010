Summary

The Creating Classes section of this lesson focused on the pedantic details of writing the classes from which objects are created. The Spot applet described at the end of this lesson is a typical (if somewhat simplistic) applet that bridged the gap between the academics and the reality of writing a class. The next lesson takes you back to school and talks about all of the features of Java that help you define and enforce relationships between classes.
