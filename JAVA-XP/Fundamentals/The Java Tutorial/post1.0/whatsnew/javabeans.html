JavaBeans(tm)

The JavaBeans architecture defines Java's reusable software component model. The JavaBeans API allows third-party software vendors (ISVs) to create and ship reusable Java components (Beans), such as text, spreadsheet, or graphing widgets, that can be composed together into applications by non-programmers.
