The third RadioShow specification

The third iteration of the RadioShow application will add two more options to the main allowing the user to resequencing the list or to remove an element from a particular location.

The menu to be presented to the user will appear as follows. 

	Radio Show Main Menu

A. List Radio Slot Items
B. Add Radio Slot at End
C. Resequence the Slots
D. Remove a Slot
E. Exit

Please enter your choice :

Resequencing a list ensures that the start time of every element in the list is set equal to the end time of the previous element, apart from the start time of the first element which is set to zero.

The remove option will ask the user to identify the slot that they would like removed by indicating its position.

When the user chooses option C the program will finish.
