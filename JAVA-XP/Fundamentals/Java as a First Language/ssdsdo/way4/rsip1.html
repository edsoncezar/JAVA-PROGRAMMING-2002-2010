The PlayTimeTerminalInput class

The PlayTimeTerminalInput class contains a single class-wide method called readPlayTime(), which will engage the user in a dialog to obtain a PlayTime instance.

A PlayTime instance is represented in its toString() method in the form 1 min 23.4 seconds. The easiest (although not the best technique) to obtain this information from the user is to ask them to input the number of minutes, the number of seconds and the number of tenths of a second in turn. For example:

Demonstrating PlayTime Input Routine

Please enter a play time 
Please enterThe number of mins   : 1
Please enterThe number of secs   : 23
Please enterThe number of tenths : 4

Thank you, you input 1 min 23.4 seconds

End of Play Time Terminal Input Demonstration

This can be accomplished by calling the ValidatedInput readLong() method three times with the appropriate prompts and arguments.

Lines 0025 to 0034 call the readLong() method three times, following which the three integer values are combined, as shown, before being used as the argument to the playTime() constructor on line 0039.

The PlayTimeTerminalInputDemonstration class contains a main() method which calls readPlayTime() to engage the user in a dialog and then outputs the value obtained.
