The fourth RadioShow design

The dispatch option of the high level design will require the addional option to be added.

 

The design of the listBySlotType() method is as follows.

The method requires theList of slots and also the typeMenu as arguments. Its first step is to use the typeMenu offerMenuAsInt() method to obtain from the user an indication of which elements they would like listed. Having obtained the typeToShow each slot in the list is considered in turn and only if it is of the typeToShow is it output.
