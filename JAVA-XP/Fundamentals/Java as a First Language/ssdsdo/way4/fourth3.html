The first step of the method, on line 0148, obtains from the user the typeToShow making use of the listMenu offerMenuAsInt() method. The for loop, on lines 0149 to 0155, then iterates along the entire list obtaining the currentSlot from the list on each iteration. The if structure then decides if the slotType of the currentSlot is equal to the typeToShow and the body of the if outputs details if so.
