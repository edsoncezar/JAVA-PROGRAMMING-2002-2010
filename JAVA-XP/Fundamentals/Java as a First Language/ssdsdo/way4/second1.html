The second RadioShow specification

The second iteration of the RadioShow application will add a third option to the main offering the user to the option of adding a RadioSlot to the end of the list.

The menu to be presented to the user will appear as follows. 

Radio Show  
Radio Show Main Menu
A. List RadioSlot items. 
B. Add Radio Slot to end of list.
C. Exit
Please enter your choice :

When the user chooses option B a dialog with the user, identical to that presented in the first part of waypoint 4, will be provided. When the user has entered the details of a RadioSlot it will be added to the end of the list. If option A is subsequently chosen then a detailed list of all slots in the list will be presented.

When the user chooses option C the program will finish.
