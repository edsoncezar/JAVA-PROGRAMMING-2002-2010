Inputting details of a RadioSlot

This part of the waypoint will combine the input operations, already introduced, to demonstrate how details of a fuel pump can be obtained from the user and used as arguments to the construction of a RadioSlot instance.

To do this the user must be asked to input the value of all four RadioSlot attributes:

  • the slotType : a manifest integer value
  • the slotDescription : a String
  • the duration : a PlayTime
  • the startTime : a PlayTime

The slotType value can be obtained from a ManifestInput instance.

The slotDescription can be obtained from a BufferedReader connected to the keyboard.

The duration and startTime values will require a specialised class wide input method, called readPlayTime() in a class called PlayTimeTerminalInput.

With this method available: the class wide readRadioSlot() method can be supplied within a class called RadioSlotTerminalInput.
