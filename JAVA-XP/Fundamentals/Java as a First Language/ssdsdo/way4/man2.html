Demonstrating the ManifestInput class

The ManifestInputDemonstration class is an interactive client for the ManifestInput class. It constructs an instance of the class with a list of the names of the colours of the rainbow (red, orange, yellow, green, blue, indigo, violet) and then demonstrates both the getManifest() and readManifest() actions.

The first part of the ManifestInputDemonstration class is as follows. 

0001  // Filename ManifestInputDemonstration.java
0002  // demonstration harness for ManifestInput class.
0003  //
0004  // Written for Waypoint 5, 97/8
0005  // Fintan Culwin, V0.1, January 1998.

0012                                            "yellow", "green",
0013                                            "blue",   "indigo",

0015

0017  
0018     ManifestInput colourInput = new ManifestInput( colours);
0019     int           colourValue = 0;

0021        System.out.println( "\n\t Manifest Input Demonstration \n");
0022  
0023        System.out.println( "Input object constructed to input the values \n" +
0024                            "red, orange, yellow, green, blue, indigo, violet\n");

Line 0007 imports the ManifestInput class and line 0011 to 0014 declares a private static final array ([]) of Strings called colours, with the seven items as listed.

Within the main() action this array is then used, on line 0018, as the argument to the ManifestInput constructor to prepare an input object, called colourInput, which is capable of inputting the names of the colours of the rainbow. The primitive int object, colourValue, declared on line 0019, will be used when the actions from the class are demonstrated. If the call of the constructor does not throw an exception then the messages on lines 0021 to 0024 will be shown to the user.
