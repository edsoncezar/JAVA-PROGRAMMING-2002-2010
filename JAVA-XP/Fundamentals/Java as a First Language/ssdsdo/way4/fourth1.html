The fourth RadioShow specification

The fourth iteration of the RadioShow application will add another option to the main allowing the user to list only those slots of a particular type.

The menu to be presented to the user will appear as follows. 

	Radio Show Main Menu

A. List Radio Slot Items
B. Add Radio Slot at End
C. Resequence the Slots
D. Remove a Slot
E. List by Slot Type
F. Exit

Please enter your choice :

When the user chooses the List by Slot Type option they will be presented with as second menu that asks the user to indicate which type of slot they would like to be listed. When they respond to this menu option only the slots that correspond to the choice indicated will be listed.
