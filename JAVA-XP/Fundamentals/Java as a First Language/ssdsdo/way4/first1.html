The first RadioShow specification

The first iteration of the RadioShow application will simply present a menu which contains two options.

The first option will allow the list of RadioSlot items to be shown to the user and the second option will cause the program to finish.

The menu to be presented to the user will appear as follows. 

Radio Show  
Radio Show Main Menu
A. List RadioSlot items. 
B. Exit
Please enter your choice :

When the user chooses option A a list of RadioSlot items, if any, will be shown to the user. As there is no way that details of any item can be entered this version will always indicate:

There are no RadioSlot details in the list.

When the user chooses option B the program will finish, showing a message as follows.

End of Radio Show.
