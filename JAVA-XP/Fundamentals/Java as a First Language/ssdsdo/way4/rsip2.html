The RadioSlotTerminalInput class

The RadioSlotTerminalInput class contains a single class-wide method called readRadioSlot(), which will engage the user in a dialog to obtain a RadioSlot instance.

A RadioSlot instance is represented in its toString() method in the form:

Advert Sid's Used Car Lot 
Start 4 min 0.0 seconds 
Duration 0 min 30.0 seconds 
End 4 min 30.0 seconds

The easiest (although not the best technique) to obtain this information from the user is to ask them to input the value of each attribute in turn. For example:
