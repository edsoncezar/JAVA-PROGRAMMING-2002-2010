Thw two way selection shown on the design is implemented in the if/ else/ end if structure on lines 0060, 0062 and 0067. If the listToShow is not empty then the for structure, on lines 0063 to 0066, will iterate as many times as there are elements in the listToShow. On each iteration the index value is output, on line 0064, and the element itself on line 0065.

The index values of the for loop will commence at 0 and finish one less than the number of elements in the list. To change this to the range 1 to the number of elements in the list, on line 0064, 1 is added to the index value as it is output. The listToShow elementAt() method takes an integer as an argument and returns the element stored at that location. On line 0065 this method is used to retrieve each element in turn from the Vector and it is immediately output.
