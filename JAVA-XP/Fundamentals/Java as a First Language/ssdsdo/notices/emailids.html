We have made it clear that communication from the teaching teams by email is essential. However a large number of identities are giving errors and mail is not getting through.

Mail sent to sbu.ac.uk will be delivered to the Vax computer, to pick it up on Unix you have to set a mail forward address on the Vax. Details on how to do this are on this CSD page.

Please ensure that email sent to your sbu address is being received by sending an email to yourself and making sure that you receive it.
