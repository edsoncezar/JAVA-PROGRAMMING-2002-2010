Waypoints 1, 2 & 3- For home use
this version Sept. 2000

 

This page supplies waypoints 1, 2 and 3 in a single zip file, called way123.zip (826K), that can be taken home on a floppy disk and installed onto your home machine. To transfer the file onto a floppy disk see the instructions on the 'Moving files' notice.

way123.zip

When you get the zip file home unpack it into a prepared directory, (called ssd?) then point your web browser at the file called schedule.html, and book mark it.

Not all of the links on this version of the waypoints will be working (eg the links to Unix information) but the substantive parts of the waypoints are there.
