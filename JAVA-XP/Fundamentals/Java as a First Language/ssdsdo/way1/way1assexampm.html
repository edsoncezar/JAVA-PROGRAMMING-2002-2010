Example Waypoint 1 Assessment

This page contains the example waypoint 1 assessment similar to the one you will be required to complete, in a workshop session, before the deadline indicated in the SSD Unit Guide.

The assessment is in three parts.

  • Your tutor will examine your logbook and award up to 10 marks
  • You will be required to take a multiple choice quiz to obtain up to 5 marks
  • The activities on this page will be assessed for up to 15 marks

Tasks:

Follow these instructions carefully.

  1. Follow the link; WayPoint1pm.java
  2. Save the file to a location (somewhere in the c:\work or d:\work folders, or the g:\drive if you have an sbu account)
  3.  Do the same for the file Mouse.java then bring up the Ready IDE and open the 2 java files you have just saved.
  4. Change the name of the file from WayPoint1pm.java to Way1pm.java
  5. Edit Way1pm.java so that it outputs your name and your student number. (You should ignore the rest of the file).
  6. Modify the file Way1pm.java to include an instance of the class Mouse and demonstrate it working by (at least) calling setCount(), and displaying something relevant to the screen.
  7. Compile and execute the java program contained within Way1pm.java to make sure it is working and save the output to a file, giving it the name Way1pm.txt
  8. Demonstrate you have been successful by emailing the contents of the output file, and the changed Java file to your tutor.
