Check Your Skills

We assume you have certain computing skills before we start. If you cannot do any of the tasks below, we strongly advise you to learn how as soon as possible
  1. Start Windows and log on to the network.
  2. Open NT explorer
  3. Use NT Explorer to copy groups of files
  4. Use NT Explorer to move groups of files from one folder to another
  5. Use NT Explorer to delete groups of files
  6. Create new directories (folders) and subdirectories
  7. Open a Browser such as Netscape or IE, access a file using a URL and save the file to your local machine
  8. Transfer (save) files on the hard disk to a floppy disk
  9. Transfer files on your floppy disk to your own folder on the Hard Disk

If you need to develop any of these skills, there is a good deal of support available to you at the Learning Resources Centre (LRC) and we urge you to take advantage of these resources at the first opportunity


There are other skills you will have to develop quickly at the start of the course. These include:
