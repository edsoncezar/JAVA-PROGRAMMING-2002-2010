Using the Compiler

First load up the Simple Java IDE by clicking on the 'Ready' icon
or from the start bar and you should find you are presented with the following screen. It has been chosen to be as simple as possible to get you programming right away


You can start typing in a program in the white edit screen immediatly, or you can load up a program from a file.

  1. Click on the OPEN button and load up the file you saved in the previous section (if you have not done that yet, go back and follow the instructions here).

    You should see the program in the editor as above

  2. Run the program (select RUN from the menu or press F1) This will compile and run the program, showing the output in a new window.
  3. Close the output window and modify the original program so that it will display 
    Goodbye friends 
from yourname
today is date

    using your name where it says yourname and todays date where it says date

  4. Save the file with a new name (Use File, SaveAs)

It is important you know what is going on here. The program hides a lot of the details.

  • The source code you loaded and edited is visible in the editor (the white screen you type into).
  • If you select Run -> compile the program converts the java source code into java byte code storing it in a .class file.
  • When you run the program, if the class file does not exist the source code is compiled into java byte code and the java class file is passed on to a java interpreter which runs it, along with other system classes, sending the results to the output window.

Instead of an executable file coming from a compiler and linker, as you would from C++ or Fortran for example, java is compiled into byte code format. these byte codes are then executed by the virtual machine. The advantage is being able to run the byte code on any machine which has a java virtual machine (JVM) available. Internet browsers such as Netscape and Microsoft Internet explorer (MSIE) have a JVM within them allowing a type of Java class called an applet to be downloaded over the internet and run locally.

We will explain programs like the above in more detail soon, but you should notice the method main() is the point where an application starts execution
