The Die Class - An answer

The Problem

We would like to create a Die class for a game. This should be created so that it will be useful for any games we might create in the future which require dice. Note:
One DIE
but two DICE

Tasks:

  1. For the Die Class, first answer the following questions
    1. What value(s) will it hold? What types of values?
      The face value of the Die and the maximum value on the face is probably the only values to be stored. The type of values need to be whole numbers of some sort. Integers would be sufficient
    2. What does an object of class Die do? What methods will it have?
      There should be a way to throw the Die and a way to get the face value. It will have to be initialised in some way
    3. Do all dice have the same number of sides for all games? What is the implication of this answer?
      No they don't all have the same number of sides! Games like Dungeons & Dragons use dice with 10, 12 or even 20 sides. The implication of this is we need to be able to state how many sides the die has before we use it. This is done in the constructor

    You might notice that I have included 2 constructors.
    The first will set the faceValue to 6 by default.
    The second will accept an integer which will be used as the maximumFaceValue









  3. Using previous examples as a starting point, write a Java class for the Die class
    4.  // The Die Class 

  public class PCDie extends Object{
      private int faceValue;
      private int maxFaceValue;

  	             //  constructor
      public PCDie(){
         maxFaceValue = 6;
         throwIt();  // give the faceValue a valid value
      }       // End constructor 

  	             // 2nd constructor
      public PCDie(int noFaces){
         maxFaceValue = noFaces;
         throwIt();  // give the faceValue a valid value
      }       // End constructor  

	             // Throw the Die 
      public void throwIt(){
          faceValue = (int)(Math.random()* maxFaceValue)+ 1;
      }        // End throwIt()

	             // a 'getter'
      public int getValue(){
          return faceValue;
      }         // End getValue() 
  }         // End class PCDie
