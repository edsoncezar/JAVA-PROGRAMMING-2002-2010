First Steps

First, a small warning about using the system.


You will be able to log on to the machines in the laboratories either as a temperary user or as yourself with your own unique login. Once you have your own login, you are working in a secure environment.

Log out of a secure machine when you are finished, or expect the next person to access to all of your files, and your email.
  • You need to create a set of directories to work in on the local machine and on your floppy disks
  • Work with files on the hard disk whenever possible and make lots of back-ups
  • There are secure drives on the network where you can store your work and these are one of the preferred areas to keep your work because it will then be available wherever you are in the University, and will not be available to other students
  • Only store work on floppies when transporting it to and from the University, or to make back-ups of your work. It is a bad idea to use a floppy disk as your default drive with the compiler or with say a Word Processor, Spreadsheet or Database. (Floppies are not as reliable as hard disks and there will come a time when the floppy becomes unreadable)...

...You have been warned.

Tasks:

  1. Log onto the system and create working directories (folders) on the disk.
  2. Create a set of directories on a floppy ready to store your work
  3. Using this browser, load up the Java source file  Hello.java
  4. Save the file into one of the directories you have prepared on the hard disk.
  5. Start up NT explorer and copy the file you have downloaded into one of the directories on the floppy disk for safe keeping
