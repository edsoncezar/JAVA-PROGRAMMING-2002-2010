Extend the ChangingTemperatureDemonstration program so that after it has warmed freezing and cooled boiling and shown the states of the instances, it then cools freezing and warms boiling. Predict what the states of the two instances should be and then execute the demonstration program to see if you are correct.

Make a list of all the methods that an instance of ChangingTemperature program has and decide is all of the methods have, or have not, been demonstrated by the ChangingTemperatureDemonstration program.

Extend the Angle class, which you should have already constructed in the previous activities, to produce the ChangingAngle class. This class hould introduce two new methods called wider() and narrower() which increase or decrease the angle by 1o.

To do this you should first produce a class diagram for the ChangingAngle class and then construct it. And then an instance diagram for the ChangingAngleDemonstration class before constructing it.
