The ChangingTemperatureDemonstration program.

The ChangingTemperatureDemonstration program will construct two instances of the ChangingTemperature class and show their state to demonstrate that they have been correctly constructed.

 

It will then call one of the instance's warmer() method and then the other instance's cooler() method in order to demonstrate that these methods appear to be operating correctly.

The first part of its implementation is as follows.

The two instances are declared, but not constructed, on lines 0014 and 0015.

Lines 0017 and 0019 output some information to inform the user of what program is running and what it is about to do.

The two ChangingTemperature instances are declared and constructed using both constructors on lines 0020 and 0021 and their state is shown on lines 0022 and 0023. So far this is very similar to the TemperatureDemonstration program and should produce the following output. 

Changing Temperature Demonstration

Construcing & showing two instances ... 

Freezing  is 0.0 degrees centigrade
Boiling   is 100.0 degrees centigrade

Line 0025 indicates what is about to happen and then line 0026 calls the freezing instance's warmer() method before line 0027 calls the boiling instance's cooler() method.

If freezing has been warmed by 0.1oC its state should now be 0.1oC; likewise if boiling has been cooled by 0.1oC its state should now be 99.9oC and lines 0028 to 0029 output the states of the two instances to show this. The output should be as follows. 

Warming freezing & Cooling Boiling ... 

Freezing  is now 0.1 degrees centigrade
Boiling   is now 99.9 degrees centigrade

Line 0031 marks the end of the main() method and line 0033 the end of the class.
