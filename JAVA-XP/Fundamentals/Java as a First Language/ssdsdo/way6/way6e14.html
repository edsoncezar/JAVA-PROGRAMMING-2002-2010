Activities for the Timer class

The output produced by the TimerDemonstration program was obtained on a desktop pc. The values output when it is executed on a much more powerful multi-user system (such as CSD Unix) may be quite different. Obtain an execute the TimerDemonstration program on different machines, try to explain the differences and, if necessary, adapt the demonstration client as needed. (Hint: a machine with a much more powerful processor will be able to count much faster.)

How accurate is the Timer? (Can you use a stop watch to find out??) Is it more or less accurate on different machines?

Each instance of the Timer class will establish its own thread. Demonstrate this by adapting the demonstration program so that is has a number of Timer instances and that they all time independently of each other.

Use the BasicMenu class to implement a stopwatch program. That is it should be possible (as a minimum …) to start the stopwatch, output the time elapsed, stop the stop watch and reset the stopwatch.
