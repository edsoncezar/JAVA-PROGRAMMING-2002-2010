Waypoint 6 part 1

This part of this waypoint is concerned with streams and files which have already been introduced in the lecture series accompanying these waypoints.

The introduction will involve redeveloping the RadioSlot and BasicTime classes from waypoint 2 to include methods to write the state of a RadioSlot to or from a file. To validate that this is working correctly the RadioShow artefact from waypoint 4 will be iterated again to allow the details to be written to or read from a file.
