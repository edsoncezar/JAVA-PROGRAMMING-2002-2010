The PlayTime class diagram

The PlayTime class diagram is as follows.

It extends the BasicTime class and introduces two new methods, timeBetween() and addTime().

The timeBetween() method takes a PlayTime instance, called startTime, and returns a PlayTime instance containing the time between the value of the current instance and the value of the startTime instance.

For example if this instance represented 8 min 35.6 seconds and the startTime instance represents 7 min 30.3 seconds, the PlayTime instance returned should represent 1 min 05.3 seconds.

The addTime() method also takes an argument, called addTo, of the PlayTime class and returns a PlayTime instance. It will add the time represented by the argument to the time represented by this instance and return the added value.

For example if this instance represented 5 min 15.1 seconds and the addTo instance represents 6 min 41.8 seconds, the PlayTime instance returned should represent 11 min 56.9 seconds.
