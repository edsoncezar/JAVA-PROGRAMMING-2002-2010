The BasicTime class diagram

The class diagram for the BasicTime class is as follows.

The class is in the playtime package and has a single attribute, duration, which is an integer representing the time in tenths of a second.

The default constructor will set the value of duration to 0 and the alternative constructor to the value of its argument. So to create an instance which represents 1 min 0.0 seconds the argument would have to be 600. (1 minute is 60 seconds which is 600 tenths of a second).

For example if an instance is constructed to represent a duration of 2 min 15.9 seconds. The argument to the alternative constructor would have to be 1359 which is 2 min (1200 tenths of a second) plus 15 seconds (150 tenths of second) plus 9 tenths of a second (1200 + 150 +9 = 1359).

The getMinutes() inquiry method will return the whole number of minutes. The getSecondsOver() method will return the number of seconds past the number of minute and the getTenthsOver() method will return the number of tenths of a second past the number of seconds past the minute.

For the 2 min 15.9 seconds instance constructed above, getMinutes() would return 2, getSecondsOver() would return 15 and getTenthsOver() would return 9.

The protected getDuration() method will return the value of the duration attribute. In the example this would be 1359.

The toString() method will return a String of the form 2 min 15.9 seconds.
