As the two classes are in the gasbill package the two source code files have to be stored in a subdirectory called gasbill. On the diagram this is shown as subdirectory of a directory called way2 (which itself is a subdirectory of javaprograms, which is a subdirectory of your home directory).

Extend the MeterReadingDemonstration program to construct a third MeterReading instance called otherReading with a made up address and reading value and show that the attributes have the correct values.

Have all the methods in the MeterReading class been demonstrated by the MeterReadingDemonstration program? If not extend the program to demonstrate the missing methods, if so explain why in your workbook, if unsure do both!

This is the class diagram of a class called VehicleOdometer which is intended to be used to represent an vehicle's registration number (as a String) and the number of milesTravelled (as an int).

Construct the VehicleOdometer class and also a demonstration program for it. The demonstration program should create at least two instances of the class and show that all methods appear to be working correctly.
