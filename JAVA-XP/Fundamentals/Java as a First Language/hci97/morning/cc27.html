Executing the ClickCounter as an application

To execute the ClickCounter as an application the ClickCounterTranslation source code file must contain a main() action, as follows. 

0120     public static void main( String args[]) { 
0121  
0122        Frame        frame      = new Frame("Click Counter Demo");
0123        ClickCounterTranslation theInterface    
0124                                = new ClickCounterTranslation();
0125  
0126        theInterface.init();
0127        frame.add("Center", theInterface);
0128  
0129        frame.setVisible( true);
0130        frame.setSize( frame.getPreferredSize());
0131     } // End main.
0132  
0133  } // End class ClickCounterTranslation.

Unlike C++ the signature of a Java main() action is strictly defined.

This action declares and initialises an instance of the ClickCounterTranslation class called theInterface which is added to a Frame instance called frame. The frame is then made visible before being resized to accommodate the preferred size of the application which it contains.

The Java Frame class provides a component supplying a top level window which can exist independently on the desktop.
