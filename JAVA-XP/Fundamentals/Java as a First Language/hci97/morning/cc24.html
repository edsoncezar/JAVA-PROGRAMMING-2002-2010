- init() action

0027     public void init() { 
0028        itsInterface = new ClickCounterPresentation( this);
0029        itsInterface.setValueDisplay( theCounter.countIsAsString()); 
0030        itsInterface.setMinimumState();

The init() action is called by the browser when it is clear for it to construct its interface. This is accomplished on line 0028 with a call of the ClickCounterPresentation constructor.

Having constructed the interface the value on its valueDisplay component is set to that obtained from the theCounter (ClickCounter) instance. The interface is then set into the MINIMUM_STATE and theState is set to reflect this.

When an applet's init() action concludes the interface will be constructed (and then the addNotify() action of the applet will be called) before being made visible to the user.
