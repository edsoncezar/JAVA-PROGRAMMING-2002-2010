Application, Presentation & Translation Layers

Following this view of an artefact with a GUI, it can be divided into three modules each implementing a different area of concern.

  • Application providing the essential functionality and (idealy) totally unaware of any user interface concerns.

  • Presentation responsible for displaying the interface to the user, obtaining their responses to it and changing its appearance when requested.

  • Translation responsible for mediating between the two other layers thus providing the behaviour of the interface.
