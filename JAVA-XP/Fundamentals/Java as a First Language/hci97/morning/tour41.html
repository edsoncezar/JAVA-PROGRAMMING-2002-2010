processMouseEvent() & actionPerformed() actions

0099  
0100        if ( event.getId() == MouseEvent.MOUSE_PRESSED) {

0106           } // End if.   
0107        } // End if.         
0108     } // End processMouseEvent.

This processMouseEvent() action differs by calling the show() action of the popup instance if the user double clicks, specifying the component and the location on the component where it should be shown. The effect is to popup the menu at the location where the user double clicked. If the user single clicks the effect is the same as that for the Doodle class. 

0080     public void actionPerformed( ActionEvent event) {
0081        
0082     String command = new String( event.getActionCommand());
0083     Color  hold;
0084     
0085        if ( command.equals( "Clear")) { 
0086           this.repaint();      
0087        } else if ( command.equals( "Invert")) { 
0088           hold = this.getBackground();
0089           this.setBackground( this.getForeground());
0090           this.setForeground( hold);
0091           context.setColor( hold);
0092           this.repaint();     
0093        } // End if.      
0094     } // End actionPerformed.

This action will be called when the MenuItems on the popup menu are activated. If the Clear item is activated then the repaint() action will cause the component's window to be cleared to the current background color of the component. If the Invert item is activated then the foreground and background colors of the component are reversed, the color of the Graphics context changed and the component is then cleared.
