The CrossDoodle classes' constructor calls its parent (Canvas) constructor and then resizes itself to that indicated by its arguments. Its only other overriding action is paint() which draws the four lines, two diagonal, one horizontal and one vertical. This action will be called every time a part of the component is uncovered. This action will be called every time a part of the component is uncovered as the scrollbars are moved.
