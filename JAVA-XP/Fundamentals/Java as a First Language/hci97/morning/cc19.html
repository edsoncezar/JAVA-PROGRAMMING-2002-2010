Layout management will be considered on the next slide.

Each button is created with a call of the Button constructor specifying the text which it is to display. Each has an actionCommand registered and all have the (cast) ActionListener value of its applet registered to receive events.

The valueDisplay is mounted on its own valuePanel and the three Buttons on the controlPanel. Both are added onto the applet's Panel.
