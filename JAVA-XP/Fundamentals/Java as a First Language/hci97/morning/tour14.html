The ItemEvent passed as an argument contains various attributes, which have different meanings depending upon which component and/ or what user action caused the event to be generated. The attributes illustrated here are itemSelectable which is the checkboxItem activated, item which is the text label of the component, stateChange which is SELECTED or DESELECTED and the event ID which is ITEM_STATE_CHANGED.

	State Change is Selected
