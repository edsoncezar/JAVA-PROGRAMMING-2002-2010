The construction of a CheckboxGroup and its association with a set of Checkbox instances causes those instances to behave as a set of radio buttons.

The same itemStateChanged() action can be used with this class and results in the same output when a button is selected or deselected. 

	Item Selectable is right Button
	Item is Right
	State Change is Selected
	ID is ITEM_STATE_CHANGED.
