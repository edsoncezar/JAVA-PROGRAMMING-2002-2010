These actions are called as appropriate when the user interacts with the component using the mouse, and implement the behaviour shown in the state transition diagram.

The processMouseEvent() action only reacts to pressing of a mouse button and stores the location of the pointer in the private lastX and lastY attributes.

The processMouseMotionEvent() action only responds when the pointer is being dragged, i.e. with the mouse button held down. It draws a line to the current location from the last known location before recording the current location as the last known location.
