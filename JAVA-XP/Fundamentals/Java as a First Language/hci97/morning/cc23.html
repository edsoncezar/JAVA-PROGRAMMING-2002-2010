This class imports the ClickCounter and ClickCounterPresentation class as it has a need to know about them. (The other two classes have no such knowledge.)

The manifest values on lines 0017 to 0020 are used to model the state transitions of the interface with theState instance variable on line 0021 recording the state of the interface.

The ClickCounter instance theCounter is declared and constructed on line 0023, with a small range of values to facilitate testing. The ClickCounterPresentation instance, itsInterface, is declared on line 0024 but it is not constructed until its init() action is called.
