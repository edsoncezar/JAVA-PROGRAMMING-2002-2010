The StikNoteWindow class supplies the main window in which the user types the message and the StikNote Button which causes it to be transferred to a StikNoteNote window. Its init() action constructs this interface by mounting a 20 column by 10 row TextArea above a Post It Button whose ActionListener is the cast value of the applet identity provided as an argument. Both of these instances are mounted within itsApplet before the action concludes.
