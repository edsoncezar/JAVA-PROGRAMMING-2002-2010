0014  class DayBox extends Canvas {
0015  
0016  private int            ordinal       = -1;

0018  private boolean        isHighlighted = false;

0020  private boolean        isPrimed      = false;
0021  private ActionListener sendToHere;
0022  
0023  
0024     protected DayBox( int            itsLocation,
0025                       ActionListener listener) { 
0026        super();
0027        ordinal = itsLocation;  
0028        this.enableEvents( AWTEvent.MOUSE_EVENT_MASK);
0029        sendToHere = listener;
0030     } // End DayBox constructor.

The six attributes, shown in the class diagram, are declared and initialised on lines 0016 to 0021. If theDayNumber has the value 0 then the box is considered inactive.

The constructor commences by calling its parent (Canvas) constructor and then records its ordinal value. Mouse events are then enabled upon the component and the identity of its listener is recorded.
