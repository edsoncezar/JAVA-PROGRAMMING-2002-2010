Retrieving properties - strategy

Establish each property as a data attribute of the client program class with a suitable default value.

For each property provide a state setting action (and a state enquiry action). The state setting action should only change the value

In the main() action extract the properties from the args[] array and call the appropriate state setting action in the client program instance, before calling the client program's init() action explicitly.

In the Applet init() action, if running as an applet, call getParameter() for each possible property and pass the value obtained to the appropriate state setting action. (If it is not specified it will be null and so will not change the default.)

Provide a private action to apply the values of all properties. These will have been set by main() from the command line if running as an application, or by getParameter() if running as an applet. Call this action from the init() action.
