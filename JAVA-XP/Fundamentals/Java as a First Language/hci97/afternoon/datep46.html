Demonstrating the I18n DatePanel implementation

Demonstration of the internationalised DatePanel Component can be accomplished by temporarily using an alternative ResourceBundle getBundle() action which takes a second argument indicating the linguistic environment to be used. For example the French language variant can be demonstrated by replacing this getBundle() call 

0075       resources   = ResourceBundle.getBundle( "DatePanelResources");

with 

0075       resources   = ResourceBundle.getBundle( "DatePanelResources", 
0076                                                     Locale.FRENCH);

Where Locale.FRENCH is a manifest value of the Locale class indicating a French linguistic environment.
