The init() action of the main application, StikNote, class constructs an instance of the StikNoteWindow class supplying itself (this) as the single argument. This will cause the interface to be constructed within itself. It will also receive ActionEvents from the interface's Post It Button which will be handled by its actionPerformed() action, as follows. 

0033     public void actionPerformed( ActionEvent event) {  
0034     
0035     StikNoteNote theNote;
0036     String     itsContents = mainWindow.getMessage();
0037  
0038        if ( itsContents.length() > 0) { 
0039           theNote = new StikNoteNote( itsContents, this);
0040           mainWindow.clearMessage();                                    
0041        } // End if.   
0042     } // End actionPerformed.

Upon receipt of an event it obtains the message from the mainWindow, using its getMessage() action and, if the message is not empty, creates a new instance of the StikNoteNote class containing the message before clearing the message from the mainWindow.
