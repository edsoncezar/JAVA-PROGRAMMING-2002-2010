The DayBox class - class diagram

The ordinal attribute is the dayBox instance's location within the iteration forming the MonthPanel and can only be set as the instance is constructed. The dayNumber attribute is the day number which the box is currently displaying and can be changed when the MonthPanel is re-laid out.

The highlighted attribute indicates if the box is selected, the bordered if it is currently under the mouse pointer and primed if the mouse button has been pressed whilst on it. The sendToHere attribute is the identity of the listener which will listen to events from this component.

All the actions are protected, apart from paint(), as they are only intended to be used by the MonthPanel which is in the same package. The paint() action has to be public in order for the run time system to use it when the box has to be re-displayed.
