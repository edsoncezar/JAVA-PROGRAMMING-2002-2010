The setDate() action commences by calling the select() action of each of the Choice components passing a manipulated year value or a pre-decremented month value. It then calls the monthPanel reConfigure() action passing on the three arguments, causing the monthPanel to lay itself out as appropriate. Its last step, on line 0101, is to call the dateSelected() action which will result in an ActionEvent being dispatched from the DatePanel.
