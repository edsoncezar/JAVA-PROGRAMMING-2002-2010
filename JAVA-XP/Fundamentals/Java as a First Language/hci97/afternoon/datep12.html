This action commences, on lines 0066 to 0071, by clearing the entire component to its background colour. If it is inactive, i.e. if theDayNumber is zero, nothing else happens.

Otherwise: lines 0074 to 0080 render theDayNumber centred within it in a manner similar to the MessageBox. Lines 0082 to 0089 and 0091 to 0096 draw the highlight and border, if they are required, respectively.
