Localisation - environment?

The System properties list can be used to obtain details about the environment:

     Properties preset = System.getProperties();
                preset.list( System.out);

Which will produce information which always includes the following: 

        java.version=1.1_Final
        java.class.version=45.3

        os.arch=x86
        os.name=Windows 95
        os.version=4.0

        file.separator=\
        path.separator=;
        line.separator=

        user.language=en
        user.region=GB
        user.timezone=GMT
        user.name=fintan

The Property list can be enquired of for a specific property, as follows: 

  String userName = new String( preset.getProperty( 
                                           "user.name"));
