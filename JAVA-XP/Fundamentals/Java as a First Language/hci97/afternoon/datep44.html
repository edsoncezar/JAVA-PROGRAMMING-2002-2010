Accessing the DatePanelResources class

The MonthPanel class has to be amended to retrieve the resources from the DatePanelResources ResourceBundle, as follows. 

0046    String         monthNames[];
0047    ResourceBundle resources;
----
0075       resources   = ResourceBundle.getBundle( "DatePanelResources");
0076       monthNames  = (String[]) resources.getObject( "monthNames");   
0077       monthChoice = new Choice();
0078       monthChoice.addItemListener( this);
0079       for ( thisOne = 0; thisOne < 12; thisOne++){ 
0080           monthChoice.addItem( monthNames[ thisOne]);
0081       } // End for.

On line 0075 the ResourceBundle called DatePanelResources is opened with its identity referenced by resources. Once opened the Object whose key is "monthNames" is retrieved from the bundle and cast to its known type array of Strings and referenced by monthNames.

Elements of this array are then used as the arguments of the addItem() action on line 0080 to install them as the Item strings of the monthChoice Choice component.
