The actionPerformed() action is called from the dayBoxes whenever they are selected by the user clicking the mouse upon them. The action commences by clearing the existing highlight, setting the highlight of and recording the identity of the box which generated the event, and then propagating the event to its own registered listener's actionPerformed() action. (This is the DatePanel instance which it is contained within which, as will be shown, propagates it further.)
