Chapter10 Summary

  • Julian dates represent calendar dates as sequential integer values, allowing arithmetical calculations to be easily made.

  • The starting point for the design of any class hierarchy can be the consideration of a standard list of action categories.

  • The six possible relational operations ( =, !=, >, <, >=, <=) can be supplied as three actions ( =, >, <) and their corresponding negations ( !=, <=, >= ).

  • Software should be designed from the outset for international use.
