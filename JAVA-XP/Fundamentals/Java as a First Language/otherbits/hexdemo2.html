Other number bases

In addition to decimal (base 10) and hexadecimal (base 16), values can be expressed in any other base. Common bases used in computing include binary (base 2) and octal (base 8).

    The binary number 11 is one set of two plus one, which is three.

    The octal number 11 is one set of eight plus one, which is nine.

    The decimal number 11 is one set of ten plus one, which is eleven.

    The hexadecimal number 11 is one set of sixteen plus one, which is seventeen.
