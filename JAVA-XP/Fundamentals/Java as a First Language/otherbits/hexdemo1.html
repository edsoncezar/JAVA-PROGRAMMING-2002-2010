Larger hexadecimal numbers

Larger hexadecimal numbers make use of more than one hexadecimal digit.

For example:

    The hexadecmal number 12 is one set of sixteen plus two, which is eighteen.

    The hexadecimal number 48 is four sets of sixteen plus eight, which is seventy two.

    The hexadecimal number 4b is four sets of sixteen plus eleven, which is seventy five.
