The default constructor, without any arguments, is declared and defined on lines 0015 to 0017. It is implemented as an indirecting call of the alternative constructor (this) passing the argument 0. The alternative constructor stores the value of its argument, initialCount, in the two data attributes: counted and theInitialCount. This ensures that upon construction a BasicCounter instance is in a well defined initial state.

Java does not have a forward reference rule and so the indirection from the default constructor to the alternative constructor on line 0016 is allowed. This would not be possible in other environments, but if Java is being used as a first language then this need not be commented upon (at this stage?).
