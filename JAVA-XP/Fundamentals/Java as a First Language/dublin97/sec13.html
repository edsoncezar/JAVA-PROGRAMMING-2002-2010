The RoomMonitor enterRoom()
and exitRoom() actions

The enterRoom() action has to record the entry by calling the count() action of the encapsulated theCounter instance. It can then increment the totalNumberEntered attribute before deciding if the number of people now in the room is greater than the maximum number of people ever in the room, and incrementing the maxEverInRoom attribute if so. It is implemented as follows:

0027     public void enterRoom() {
0028        theCounter.count();
0029        totalNumberEntered++;
0030        if ( this.numberCurrentlyInRoomIs() > this.maxEverInRoomIs()){ 
0031           maxEverInRoom++;
0032        } // End if.
0033     } // End enterRoom.

The exitRoom() action is considerably simpler calling the unCount() action of the encapsulated theCounter instance, as follows: 

0036     public void exitRoom() { 
0037        theCounter.unCount(); 
0038     } // End exitRoom.
