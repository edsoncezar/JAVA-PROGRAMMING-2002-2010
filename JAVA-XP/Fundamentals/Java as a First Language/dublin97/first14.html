Practical activities

Make a list of all public actions which an instance of the LimitedCounter class will supply. Adapt the LimitedCounter class file to make the constructors and the class public and then adapt the BasicCounterDemonstration class to demonstrate the effectiveness of the implementation. (Hint: the line: System.out.println( aLimitedCounter.isAtMinimum()); will output true or false as appropriate.

Extend the LimitedCounter class to produce a public SteppingCounter class. This class has an additional attribute called countBy which determines the change in the counted value of the count() and unCount() actions. For example if the value of countBy is 3 and the current value returned by numberCountedIs() is 10 then the effect of the count() action will be to change this value to 13.

Hint 1: to accomplish this the for loop construct will be required, possibly as follows:

	for( index = 0; index < countBy; index++) { 
	   // do something;
	} // End for.

Hint 2: The count() and unCount() actions will need to be overidden.

Demonstrate the effectiveness of the implementation.
