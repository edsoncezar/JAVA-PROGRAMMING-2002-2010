Lines 0001 to 0006 are the who, what, why, where, when comments which are included at the start of every source code file and will not be mentioned again.

Line 0008 identifies this file as being contained within the Counters package of classes and line 0010 declares the BasicCounter class as an abstract class which extends the Java Object class.

The two encapsulated data attributes are declared, with private visibility, as instances of the primitive int type having the default value 0. (Unlike most other environments Java defines default values for all instances which for instances of the int type is zero. However it does no harm to make this explicit.)
