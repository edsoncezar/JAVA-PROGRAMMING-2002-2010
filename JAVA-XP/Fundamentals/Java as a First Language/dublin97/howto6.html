Compiling

Compilation can now be accomplished by entering the compilation command javac %f in the Command text box of the Execute Dos … dialog.

 

The %f indicates that the full filename, including the extension, of the file in the currently active window should be used. Other possibilities include %n which indicates the filename without the extension and %p the directory path. Pressing the OK button will launch a DOS shell, execute the command and capture the output into a CommandOutput window.
