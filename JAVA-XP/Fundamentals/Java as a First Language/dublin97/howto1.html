Developing in Java - overview

 

 

This part of the tutorial will introduce and describe an environment, for use on a PC with Windows 95 or NT, which can be used for the development of Java artefacts. The environment is Sun's Java Development Kit (JDK) (version 1.1.X) and the Programmer's File Editor (PFE) (version 32 bit 0.06.002). These tools were chosen as they are both free of charge and so can be made available to students for use on their home machines.

The current version of the Sun JDJ can be obtained from http:\\www.sun.products\java. A 1.1.X version should be used and any 1.0.X versions should be regarded as obsolete and disposed of. To find out which version is running, from the MS/Dos command line type: C>java -version.

The current version of PFE can be obtained from http:://www.lancs.ac.uk/people/cpaap/pfe/.

This tutorial will not describe the installation of these tools. The installation instructions distributed to students at SBU is included in Appendix A.
