The RoomMonitor toString() action

If a class declares a public action called toString() which takes no arguments and returns a String value, then this action will be automagically called when an instance of the class is supplied as an argument to a call of System.out.println(). The implementation of the toString() action for this class is as follows. 

0056     public String toString() {
0057        return "Now in room : "    + this.numberCurrentlyInRoomIs() + 
0058               " Max in Room : "   + this.maxEverInRoomIs()         +
0059               " Total entered : " + this.totalNumberEnteredIs();
0060     } // End toString.

The implementation makes use of the String catenation facility (+) to compose the String value to be returned. By another automagical process whenever an int value is catenated to a String its value is automatically supplied as a String.

If a newly constructed instance of the RoomMonitor class is supplied as an argument to System.out.println(), as follows:

0018     WarningCounter demoCounter = new WarningCounter();
0019        System.out.println( demoCounter);

the output produced would be:

Now in room : 0  Max in Room : 0  Total entered : 0
