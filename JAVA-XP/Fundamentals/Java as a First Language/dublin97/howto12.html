PFE miscellaneous

Select all text between matching braces.

<CTRL><SHIFT>W will select all the text between the nearest opening and closing brace. Pressing again will select the next pair of enclosing braces. This is useful for cutting and pasting or checking that a closing brace actually matches the opening brace.
