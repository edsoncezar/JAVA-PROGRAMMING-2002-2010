LimitedCounter - other actions

The two actions associated with the minimumCount attribute are implemented as follows. 

0032    public boolean isAtMinimum() { 
0033       return this.numberCountedIs() == this.miniumumIs();
0034    } // End isAtMinimum.


0041    public int minimumIs() { 
0042       return miniumumCount;
0043    } // End minimumIs.

Line 0033 uses the minimumIs() action, supplied by the LimitedCounter class, and the inherited numberCountedIs() action, supplied by the BasicCounter class. It could be implemented as 

0033       return this.numberCountedIs() == miniumumCount;

Making direct reference to the instance attribute minimumCount. This should not be encouraged and the use of actions available to instances of the class wherever possible should be insisted upon. It would not be possible to implement line 0033 as follows: 

0033       return super.counted == miniumumCount;

As, although a counted instance attribute is inherited by LimitedCounter instances it was declared as a private attribute of the BasicCounter class and is only visible within that scope.
