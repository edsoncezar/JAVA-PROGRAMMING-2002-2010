The RollOverCounter constructors

As the RollOverCounter has no instance attributes it need not initialise any state within itself and the constructors are implemented as indirecting calls of its parent (super) constructors.

0008 package Counters;
0009 
0010 public class RollOverCounter extends LimitedCounter { 
0011 
0012    public RollOverCounter() { 
0013       super();
0014    } // End default Constructor.
0015 
0016    public RollOverCounter( int minToCount, int maxToCount) { 
0017       super( minToCount, maxToCount);
0018    } // End principal Constructor.

As suggested in the previous section it seems more natural for the default constructor to appear before the alternative constructor, as suggested in the class diagram.
