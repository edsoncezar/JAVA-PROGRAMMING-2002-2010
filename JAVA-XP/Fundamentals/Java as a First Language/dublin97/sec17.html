Practical Activity

Adapt the RoomMonitor class so that a limit can be placed upon the number of people in the room. Use the demonstration harness to show that the change has been effective.

A RoadMonitor class will only allow a certain amount of traffic on a section of road, a car ca be counted as 1 unit, a goods vehicle as 5 units and a bus as 10 units. Implement a RoadMonitor class which reports upon the number of each type of vehicle currently in the road section, the maximum number of each type of vehicle in the section, the total number of each type of vehicle ever in the section and which throws an exception if the capacity of the road section is ever exceeded.

Design, implement and demonstrate a PetrolPump hierarchy. The base class might only indicate if the Pump is switched on or off, child classes might introduce attributes for the total amount of fuel dispensed (in litres), the price of the fuel and the type of fuel dispensed. Hint 1: The types of fuel can be provided as public class wide constants. Hint 2: A toString() action can be implemented as return super.toString() + "these details" .
