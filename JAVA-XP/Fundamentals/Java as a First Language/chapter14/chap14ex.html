Exercises for Chapter 14

    14.1 Complete the demonstration harness clients for the Queue, Stack and OrderedList classes and use them to increase your confidence that the classes are implemented correctly.

    14.2 Produce a black box test plan for either the Queue or Stack classes and compare it with the actions you have included in demonstration harnesses from Exercise 14.1. By how much has this increased your confidence in the implementation of the class?

    14.3 Revisit the ContactList application from Chapter 13 and re-implement it making use of an OrderedList. From the user's point of view are there any differences?

    14.4 Using the LibraryBook hierarchy from Exercise 13.6 re-implement the LibraryList application making use of an OrderedList.

    14.5 Produce a black box test plan for the applications from 14.3 and 14.4. Compare the two test plans and decide which aspects of the tests are testing the OrderedList functionality. What would this imply for the test plan of a third application which also makes use of an OrderedList?
