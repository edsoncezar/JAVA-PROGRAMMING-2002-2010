Summary of Chapter 2


  • Class declarations can include data attributes as well as actions.

  • Actions can be instance actions or class wide actions.

  • Class wide actions are usually called using the name of the class, instance actions are called using the name of an instance of the class.

  • Data attributes can be instance attributes, class attributes or class constants.

  • Every instance of a class has its own copy of instance attributes, but there is only one copy of class attributes or class constants per class.

  • Data attributes are usually declared totally private.

  • Constructors are used to place a class instance into a well defined initial state.

  • Numeric values, without any decimal parts, can be represented by variables of the primitive int type.
