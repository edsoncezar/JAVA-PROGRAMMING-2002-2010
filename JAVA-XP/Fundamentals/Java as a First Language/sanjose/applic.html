Java across the curriculum

This activity required the attendees to suggest, and subsequently evaluate on a scale from -5 to +5, areas in the curriculum where java might be suitable for use. These results are preseneted in descending sequence of popularity.

The values given for each area are the number of attendees who expressed an opinion, the avreage score and the standard deviation. This last value indicates the degree of agreement with lower values indicating a greater degree of agreement.

Notes

The distinction between a base langauge and a first language is that a first language is intended to be used for inital pedagogic purposes and not to be used thereafter, whereas a base language is suitable for use as a first language and for further use thereafter.

The inclusion of artifical intelligence indicated that Java is not a panacea for all parts of the curriculum. However the raw data indicated a halo effect with some attendees giving Java a high score for this aspect. This observation is confirmed by the high standard deviation associated with this potential use.
