Notes

JDK is admittedly not an IDE but a command line environment.

The popularity of Symantec Cafe might be as a result of its free distribution with the Core Java book, which was reported as being the most popular book resource.

The low number of attendees voting for any of these resources, especially the JDK, brings into question the amount of personal practical experience of Java which the attendees had had.
