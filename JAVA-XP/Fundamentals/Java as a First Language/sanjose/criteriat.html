The summary criteria table is as follows.

The second column, labeled n, indicates the number of respondents. The next column is the raw evaluation of the language against the criteria, with a higher value indicating greater conformance. This is followed by the standard deviation of this value which indicates the degree of consensus with smaller values indicating closer agreement. The final column is the weighted evaluation where the raw evaluation is multiplied by the weightings established from the criteria table.

The raw and weighted results both give the same relative outcomes. The most favored language is Java followed by Ada, Eiffel and Smalltalk with C++ as the least favored.
