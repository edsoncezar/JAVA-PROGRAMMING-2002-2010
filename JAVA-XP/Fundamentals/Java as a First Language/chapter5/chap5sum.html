Chapter 5 Summary

  • A child class extending its parent establishes an is a relationship.

  • As a child class cannot remove actions from a parent, which may cause the child to have inappropriate or dangerous actions, this would suggest a has a relationship.

  • A class can override the toString() action in order to define what output will be produced when it is passed as an argument to the System.out.println() action.

  • Attribute enquiry actions should be used in the implementation of the class, rather than using direct access to the attributes, to assist with maintenance and demonstration.
