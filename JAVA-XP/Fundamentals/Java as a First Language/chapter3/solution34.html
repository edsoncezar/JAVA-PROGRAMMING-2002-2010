Exercise 3.4

A specialised vehicle counter is required which counts 1 for every motorbike which passes, 2 for every car, 5 for every bus and 10 for every van. Include this class called GreenVehicleCounter into the class hierarchy.

The GreenVehicleCounter class must extend the LimitedCounter class in order to make use of the actions of the LimitedCounter class and the BasicCounter class.

The GreenVehicleCounter class can be designed with four actions, one for each category of vehicle, all of which will increment the counted data attribute accordingly.

After the expected header comments, the GreenVehicleCounter extends the LimitedCounter class : 
public class VehicleCounter extends LimitedCounter{

Several manifest values for the different categories of vehicles will be required.

private final static CYCLE_COUNT = 1;
private final static CAR_COUNT   = 2;
private final static BUS_COUNT   = 5;
private final static VAN_COUNT   = 10;

The class will require two constructors : 

   public VehicleCounter() { 
      super();
   } // End default constructor.


   public VehicleCounter ( int minToCount, int maxToCount) { 
      super( minToCount, maxToCount); 
   } // End alternative constructor.

The next four actions are required to make use of the counted() action depending on the category of vehicle is being recorded : 

   public void countCycle() { 
      this.count();
   } // End countCycle. 


   public void countCar() { 
      this.count(); 
      this.count();
   } // End countCar. 


  public void countBus() { 
     for ( int index = 0; index < BUS_COUNT; index++) { 
        this.count(); 
     } // End for.
   } // End countBus. 


   public void countVan() { 
     for ( int index = 0; index < VAN_COUNT; index++) { 
        this.count(); 
     } // End for.
   } // End countVan.
