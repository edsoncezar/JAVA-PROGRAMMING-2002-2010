Exercises for Chapter 3

  • 3.1 Produce the design of the partial LimitedCounterDemonstration client.

  • 3.2 Complete the design and implementation of the LimitedCounter demonstration.

  • 3.3 Design a class hierarchy for Noah, whose click counter as he counted the animals into the ark incremented by 2 every time he pressed it.

  • 3.4 A specialised vehicle counter is required which counts 1 for every motorbike which passes, 2 for every car, 5 for every bus and 10 for every van. Include this class called GreenVehicleCounter into the class hierarchy.

    A solution is available.

  • 3.5 Extend the Employee class from Exercise 2.4 to implement a SalariedEmployee class which has an attribute and enquiry action for the annual salary and a WagedEmployee class which has an attribute and enquiry action for the hourly rate of pay. Design, build and demonstrate these two classes; making them member classes of an Employee package.
