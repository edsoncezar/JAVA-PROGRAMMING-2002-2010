A Taste of Java is the name of an introductory talk about Java.

The text of the talk is available in the following formats:

  • divided up into several WWW pages with coloured backgrounds that can be presented a screenful at a time;
  • divided up into several WWW pages with default backgrounds that can be presented a screenful at a time;
  • as a PostScript file for A4 (about 562K bytes).
The text was last updated on 21st January 1997.
