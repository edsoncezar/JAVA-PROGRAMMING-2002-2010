Summary

Java provides a powerful set of classes in the package java.net that can be used for building useful applications that can perform network oriented tasks. You learned to use the classes Socket and ServerSocket to build a client/server program that uses the TCP/IP protocol. You also saw examples of using DatagramPacket and DatagramSocket for sending and receiving UDP datagrams. The URL and URLConnection classes can be used to access Web resources in Java applications and applets. You also learned how a Java applet can interact with the Web browser and pass URL information to the Web browser for rendering.
