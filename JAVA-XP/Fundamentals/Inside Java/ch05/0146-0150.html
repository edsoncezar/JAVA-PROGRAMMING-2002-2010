Using Basic I/O Classes

Java contains a very rich set of classes to perform input/output operations in a flexible manner. In this section, only two classes—the FileOutputStream and PrintStream classes are discussed. These classes are sufficient to perform basic input/output operations. All the Java input/output classes are in the package java.io.

You have used the static variables System.out and System.in in earlier examples to perform output and input operations. The System.out variable typically is used to print to the display device, but it is in no way restricted to output data to the display device only. You could send the output to a text file, for example. The static variables System.out and System.err are instances of the PrintStream class. The System.err class is used for sending error messages.
