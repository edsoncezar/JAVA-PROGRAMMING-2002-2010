The Integer, Long, Float, and Double Type Wrapper Classes

The Integer, Long, Float, and Double type wrapper classes deal with numeric values. Besides the common methods for type wrapper classes, the number type wrappers provide other common methods and definitions that are common to these classes: 

public final static datatype MIN_VALUE;
public final static datatype MAX_VALUE;
public static classtype valueOf(String s);

The MIN_VALUE and MAX_VALUE are defined as constants that contain the minimum (closest to zero) and maximum values of the datatype (int, long, float, double).

NOTE
Java does not have special type wrappers for byte. Short data types such as these can be handled by the Integer type wrapper, for the most part.

The valueOf() method is used to return a type wrapper class object for a given string numeric value. Note that the type returned is classtype, which is the type of the wrapper class. In addition, the Integer and Long classes have another variation of the valueOf() method that includes a second argument, which is the radix of the number contained in the string numeric value: 

public static classtype valueOf(String s, int radix);

An example of the use of valueOf is when you have a string representation of a number, and you want to convert this to the type wrapper class: 

Long xval = Long.valueOf("314159265");

Suppose that you want to convert the string representation of an int, you can use the following:

The Float and Double type wrapper classes contain the following definitions that are particular to the floating point classes only: 

public final static datatype NaN
public final static datatype NEGATIVE_INFINITY
public final static datatype POSTIVE_INFINITY
public boolean isInfinite()
public static boolean IsInfinite(datatype v);
public boolean isNaN()
public static boolean IsNaN(datatype v);
public int intValue();
public long longValue();

The NaN variable defined in public final static datatype NaN is the representation of the Not a Number value discussed in Chapter 3, "Experimenting with Java Data Types and Variables." The value of datatype can be either a float or double.

The NEGATIVE_INFINITY and POSITIVE_INFINITY contain the special values for minus infinity and plus infinity, respectively. See Chapter 3 for a discussion of these concepts.

The method isInfinite() is used to check if the value of the type wrapper object is infinite, and the isNaN() method is used to check if the value is NaN. Note that both static and non-static versions of these methods exist. For the non-static method you must invoke it for an instance of the object. These methods are therefore called instance methods.

NOTE
In Java, static methods of a class are called class wide methods. Methods that are not static are called instance methods because they need to be invoked for each instance of the class object.

The intValue() and longValue() are also instance methods that can be used with a Float or Double object to convert its value into an equivalent int or long type.
