The following example illustrates the use of the StringBuffer class. 

StringBuffer svar = new StringBuffer("Darjeeling");

svar.append(", Palace of Light");
// svar is set to Darjeeling, Palace of Light

svar.insert(12, "Lake Mansa, ");
// svar is set to Darjeeling, Lake Mansa, Palace of Light

svar.setCharAt(20, `a');
svar.setCharAt(21, `s');
// svar is set to Darjeeling, Lake Manas, Palace of Light

int len = svar.length();

Summary

In this chapter, you have learned to use Java objects. Java objects are instances of Java classes. You have learned how to access the methods of a class. Java methods can be static or non-static. Static methods are class wide, and non-static methods are associated with each object instance of a class.

You were presented with numerous example of how objects can be used. The examples were based on predefined Java classes, such as Date, PrintStream, and so on. Because of the way methods are exposed for use in Java classes, you can begin to use them without knowing the details of how the class is built.
