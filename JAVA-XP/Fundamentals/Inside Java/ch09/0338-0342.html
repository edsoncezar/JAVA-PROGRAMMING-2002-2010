If you need additional information on a font, you can create a FontMetrics object by a call to getFontMetrics() inherited from the Component class, and query this object with methods from the FontMetrics class. The following code fragment, for example, returns the number of pixels in a specified string for a specified font: 

Font fnt = new Font("Times Roman",
         Font.BOLD+Font.ITALIC, 20);
FontMetrics fm = getFontMetrics(fnt);
String msg = "To Be or Not to Be!";
// Number of pixels in message.
int pixels = fm.stringWidth(msg);
