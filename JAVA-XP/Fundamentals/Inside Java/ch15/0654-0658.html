The ScopeFrame class in listing 15.2 contains an application that was developed to display the Bean so that the user can interact with it. ScopeFrame is not a builder tool, so it doesn't do any introspection or display property sheets. This application is, however, an example of how easy it can be to work with Java Beans inside an appli-
cation.
