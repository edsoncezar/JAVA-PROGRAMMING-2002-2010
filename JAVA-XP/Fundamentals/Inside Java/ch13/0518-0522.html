Checkbox and CheckboxGroup

Check boxes are useful GUI elements when you want to present the user with a true or false choice. You could use a check box, for example, in a "Find" dialog box to ask the user if the text to be found must match the case of the entered text. Check boxes can be grouped together into a CheckboxGroup, which presents a set of mutually exclusive choices to the user. When used in this way, check boxes look and behave like what are commonly referred to as radio buttons.

The following example creates three check boxes, labeled "Coffee," "Tea," and "Milk:" 

Checkbox cb1 = new Checkbox("Coffee");
Checkbox cb2 = new Checkbox("Tea");
Checkbox cb3 = new Checkbox("Milk");
add(cb1);
add(cb2);
add(cb3);

The following code builds on the preceding example by creating three mutually exclusive check boxes (also known as radio buttons), with the last check box checked. Note that the constructor used in this example was added in JDK 1.1; the similar constructor that appeared in JDK 1.0.2 has been deprecated.

NOTE
This is only one of the many constructors and methods new to JDK 1.1. Some of the new constructors and methods provide totally new functionality, and some replace older, deprecated constructors and methods. Deprecated, as referred to in the JDK documentation, means that a constructor or method has been replaced, and will be phased out in a future version of the JDK. 

CheckboxGroup grp = new CheckboxGroup();
Checkbox cb1 = new Checkbox("Coffee", false, grp);
Checkbox cb2 = new Checkbox("Tea", false, grp);
Checkbox cb3 = new Checkbox("Milk", true, grp);
add(cb1);
add(cb2);
add(cb3);

As you are aware, the Checkbox component may be configured to restrict the user to a single, mutually exclusive choice, or it can be configured to enable multiple selections. Similarly, the List component also allows for single or multiple selection, as the following section details.

List Components

A List—commonly referred to as a list box—displays a scrolling list of choices to the user. A list box can be configured to accept either multiple selections or one selection, with one selection being the default. The following example creates a mutually exclusive list box with four entries and three visible lines. A scrollbar is automatically displayed for scrolling through the entire list: 

List suits = new List(3);
suits.add("Clubs");
suits.add("Hearts");
suits.add("Spades");
suits.add("Diamonds");
add(suits);
