Summary

Java provides a built-in mechanism for defining input/output by using predefined classes that you can instantiate or extend to provide additional functionality. In this chapter you learned the basic methods provided by the InputStream class and the OutputStream class. These classes serve as the parent superclass of the classes defined in the package java.io that are used for performing input/output. You were presented with a guided tour and examples of using the different input and output stream classes that are subclassed from InputStream and OutputStream.
