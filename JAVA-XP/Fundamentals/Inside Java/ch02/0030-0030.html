Summary

In this chapter, you learned to write a simple Java program. You also learned to create a Java text file and use the javac compiler to compile the program.

You also learned how to write a more complex program that used local variables, assignment statements, a while loop, and print statements to display a table of Centigrade values and the equivalent Fahrenheit values.

The goal of using simple sample programs is to show you a quick overview of the basic elements of the language without overwhelming you with the details and the power of the Java language. Later chapters will cover more complex concepts.
