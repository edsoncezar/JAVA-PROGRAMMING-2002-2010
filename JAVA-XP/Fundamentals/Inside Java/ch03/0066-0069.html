The conditional operator ?: evaluates an expression based upon the value of a Boolean expression. This is a ternary operator, as it involves three operands whose general syntax is as follows:

BooleanExpr ? TrueExpr : FalseExpr

If the BooleanExpr evaluates to true, TrueExpr will be evaluated, and it will become the value of the overall expression. If the BooleanExpr evaluates to false, FalseExpr will be evaluated, and it will become the value of the overall expression.

Here is an example of the use of the conditional operator: 

long   a;
long   b;
long   max, min;

max = (a > b) ? a : b;
min = (a > b) ? b : a;

In the first assignment statement, if a > b the value of the expression is a; otherwise it is b. Notice that the conditional operator is used here to find the maximum value of two numbers. Similarly, in the second assignment statement, if a > b the value of the expression is b; otherwise it is a. In this case, the conditional operator is used to find the minimum value of two numbers.

Performing Math Calculations

In many languages, mathematical calculations involving trigonometric and other mathematical functions are carried out by library routines that are either defined in the language or have to be written separately. In Java, all the standard mathematical functions are static methods in the Math class. So these can be called directly by using a suitable syntax for calling a static method of a class.

The Math class is in the package java.lang. You will learn more about packages in later chapters, but for now a brief introduction to packages will be helpful. A package is a collection of Java classes. You have learned about Java classes in Chapter 2, "Getting Started with Java." They are essentially the syntax used in Java to describe the type of an object that consists of data and methods that operate upon the data. When you write large Java programs, you can define your own classes and put them inside packages for ease of managing classes that perform similar or related functions. Java comes with its own predefined classes that are defined in packages. These packages have standard names. One of these packages is the java.lang package. Normally, to use a package in your Java program, you have to use the import statement, as shown in the following example: 

import java.lang.*;
class ImportExample
{
      // Contents of the class ImportExample
      //
      ...
} // end ImportExample

The use of the import statement with an asterisk (*) following the package name makes all classes defined in the package java.lang available for use in the class ImportExample. If you wanted to just access a specific class, such as the Math class, you would use the following import statement: 

import     java.lang.Math;

By default, the Java compiler automatically includes the import java.lang.* statement, if one has not been explicitly provided by you. Therefore, the following definition of the ImportExample class is equivalent to the previous one because the java.lang package is implicitly imported. 

class ImportExample
{
      // Contents of the class ImportExample
      //
      ...

} // end ImportExample

You can, therefore, access the method abs() in the Math package by using the following syntax: 

int     x = _3;
int     y;
y = Math.abs(x);

To create the Java API documentation for the Math class:

  1. Open a command prompt window.
  2. If you have not already done so, unzip the src.zip file located in the /java/src directory. This file contains all the source code for the standard java classes.
  3. Change to the directory in which you unzipped the java source files.
  4. Change to the /src/java/lang directory.
  5. Type javadoc java.lang.math.
  6. You can now load the java.lang.math.html file into your favorite HTML viewer.
