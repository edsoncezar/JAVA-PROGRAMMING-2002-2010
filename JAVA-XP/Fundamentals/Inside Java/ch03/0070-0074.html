As an example of how the Math class can be used, examine listing 3.5. This listing generates the dimensions of two of the sides of a right-angled triangle (see fig. 3.14) randomly using the Math.random() function listed in table 3.10. The sides of the triangle vsize and hsize in figure 3.14 are generated randomly. The random() function returns a value between 0 and 1. In the program listing, this value is scaled by a factor of 10, so that the maximum vertical and horizontal sides will be 10 units long.

The formulas for computing the dimensions of the triangle are also listed in figure 3.14. If you are familiar with a little trigonometry and geometry, you can readily verify these formulas. If not, don't worry—it will not detract from your understanding of the Java language.

Summary

In this chapter you were introduced to the concepts of types and variables, and you saw examples of how to use Java's basic types such as byte, short, int, float, double, char, Boolean, and string literals. You learned about the range of legal values that are permissible for each of the data types and how they are represented internally. You also learned about the operations that can be performed on the basic types. Several arithmetic operations that can be performed on integers such as shifts, bitwise operations, modulus, and so on, were discussed.

You learned about the unicode representation of characters in Java that enables it to support non-English languages. You were presented with examples of how you can mix and match data types and the type of the result that is generated. Many examples of the use of data types, and variables were presented.
