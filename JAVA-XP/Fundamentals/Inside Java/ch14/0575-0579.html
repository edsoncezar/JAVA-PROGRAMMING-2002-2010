Understanding AWT Menus

Two types of AWT menus exist:

  • Menu bar menus reside in the menu bar of a frame.
  • Pop-up menus, sometimes referred to as context menus, which were introduced in JDK 1.1.

An application usually has only one menu bar menu that generally appears in the main frame of the application. Because a menu bar requires a frame, an applet can't have this kind of menu unless it creates a frame with which the menu can be associated. Figure 14.2 shows an example of a menu bar menu.

Pop-up menus are becoming very standard. Users now expect that a pop-up menu will appear when they point at a user interface element and click the correct mouse button. This menu contains menu items pertinent to the element at which they are pointing. A pop-up menu associated with a selection of text in a word processor, for example, may have a menu item that opens a font selection dialog box. Figure 14.4 shows an example of a pop-up menu.

The classes that implement menus are all part of the java.awt package, and are derived from the java.awt.MenuComponent class. Figure 14.1 shows the hierarchy of classes derived from the Component class.

Each of the classes in this hierarchy are discussed in the following sections. These classes will be presented by loosely following the class hierarchy, but we will deviate slightly so that things appearing on menus (MenuItem and CheckboxMenuItem) are discussed after the menus themselves (Menu and PopupMenu).

The MenuComponent Class

All menu-related classes are derived from the MenuComponent class, which contains methods common to all menu-related classes. One such method is setFont(), which applies to any menu component.

The MenuBar Class

The MenuBar class contains methods that manage the menu bar. The add() method, for example, adds a menu to the menu bar, and the getMenuCount() method returns the number of top-level menus that a menu bar has. The PopupMenu Class

The PopupMenu is a specialization of the Menu class. Pop-up menus are used to supply context-sensitive menus within an applet or application. The PopupMenu menu class contains the show() method because a pop-up menu is not visible until the user requests it. For a complete list of constructors and methods in the PopupMenu class, refer to the following Web page: 


http://java.sun.com/products/JDK/1.1/docs/api/java.awt.PopupMenu.html

The MenuItem Class

The MenuItem class contains methods that manage a particular menu item. The setLabel() method, for example, specifies the String to be displayed in the menu item, and the setEnable() method enables and disables a menu item. Notice that the Menu class is derived from the MenuItem class: This is because a menu within a containing menu is also a menu item of the containing menu. Table 14.3 shows some of the constructors and methods of the MenuItem class. For a complete list, refer to the following Web page:
