Passing Parameters to Applets

When you write Java applications, you can pass parameters via the argument to the main() method. These parameters are specified on the command line (see the ArraySort program example in Chapter 5, "Using Java Objects and Arrays"), and they are usually designed to control the behavior of the program.

Suppose that you want to change the behavior of the applet by passing a parameter to the applet. How can you accomplish this task? You cannot pass parameters to the main() method because applets do not have a main() method. Instead, you pass parameter values through the HTML document that specifies the applet location. The parameter values to an applet are passed through the following special parameter tag: 

<PARAM NAME=name VALUE=value>

The name is the name of the parameter, and value is its value. If, for example, you wanted to draw the string in the previous HelloAppletWorld1 example in a different color or size, you could define the following parameters: 

<PARAM NAME=COLOR VALUE="Magenta">
<PARAM NAME=SIZE VALUE="36">

If you were animating an applet and wanted to control its speed, you could define the following value: 

<PARAM NAME=speed VALUE="5">

Here the value of the speed parameter has a meaning that is internal to the applet program.

The parameters to the applet must be specified between the APPLET tag: 

<APPLET CODE="location" WIDTH=400 HEIGHT=300 ALIGN=LEFT>

<PARAM NAME=COLOR VALUE="Red">

The next step is to obtain the parameter values from the HTML document into the applet code. This step is typically performed as part of the initialization of the applet in the method init().

Remember that the definition of the Applet class includes a method called getParameter(). This method is inherited by the class that you extend from the Applet class. You can specify the parameter name as an argument to getParameter(), which will then return the value of the parameter as a string object. The parameter name must match exactly (including case) the parameter name specified in the <PARAM> tag, otherwise a value of null is returned.

To read the parameter values of COLOR and SIZE described previously, for example, you can use the following: 

class MyApplet extends java.applet.Applet
{

      String color;
      String size;

      public void init()
      {
         color = getParameter("COLOR");
         size = getParameter("SIZE");

      } // end init

      // Other methods
} // end MyApplet
