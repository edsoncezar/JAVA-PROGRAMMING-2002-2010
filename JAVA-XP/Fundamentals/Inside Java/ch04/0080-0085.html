A block statement is a grouping of statements within { } brackets. Examine the method main() from the previous chapters. It has the following general syntax: 

public static void main(String args[])
{
   // statements
}

The block statement is shown in bold, and it consists of statements. These statements can be definitions of Java basic types, assignment, control flow, or even another block statement. A block statement can occur wherever a statement can occur in a Java program. A block statement is also called a compound statement.

In listing 4.1, x and y on lines 13 and 14 are defined and initialized at the start of the main() method. The variables, such as x, y, and z have a scope. The scope is the region of a program over which the variable can be referenced by its name. The scope of a variable is the region of program text from its of definition to the end of the enclosing block. Therefore, the point of x is from lines 13_33, and the scope of y is from lines 14_33.

Lines 18_27 define a block statement. Within this block statement is a declaration for z on line 19. The scope of z is therefore from line 19 to the end of the enclosing block on line 27. The variable z cannot be referenced outside this block. An attempt to reference z outside the block on line 31 will produce a syntax error. You can verify this by compiling this program contained in file BlockStmt.java on the CD-ROM that accompanies this book. The if Statement

The if statement is the most basic form of statement to execute statements based upon the evaluation of a conditional expression. In the if statement, a Boolean expression is evaluated. If the result of the Boolean expression is true, a certain
group of statements is executed; if the result is false, another group of statements is executed.

The following example shows the general syntax of the if statement: 

if (booleanExpression)

      statement-true

[else

      statement-false]

The booleanExpression is any of the Boolean expression types discussed in Chapter 3. The statement-true is any legal Java statement including the block statement, and it is executed if booleanExpression evaluates to true.
