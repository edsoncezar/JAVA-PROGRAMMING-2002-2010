The do-while Statement

The do-while statement is another type of loop statement. Recall that in the while and for statements, the Boolean test is performed at the top of the loop. In the do-while statement, the Boolean test is performed at the end of the loop. The general syntax of the do-while statement is shown next: 

do
        statement
while  (boolExpr);

The statement can be any Java statement, including the block statement. The boolExpr is the test condition for the loop. The statement is executed first. Next, the Boolean expression, boolExpr, is evaluated. If this evaluates to true, control is transferred to the top of the loop. If the Boolean expression evaluates to false, the loop is terminated and control is transferred to the end of the loop.

The break Statement

In the discussion on switch statements, you saw how the break statement can be used to exit out of a switch statement. The break statement can be used to exit any block, not just a switch statement. A break statement is typically used in a for, while, do-while loop to exit the innermost loop. Thus, if you have nested loops, a break statement in any loop will only exit the innermost loop in which the break statement occurs.

The following is the general syntax of the break statement: 

break [label];

The label is optional. Without the label, the innermost enclosing block is exited. You can use the label to break out of several nested loops.

Consider the following example of using a break statement: 

for (int x = 0; x < 1000; x++)
{
      y += x*2;
      if  (y % 500 == 333)
      break;

} // end for

The preceding for loop would normally exit when the value of x becomes 1000. A test condition, however, is performed in the if statement. If this condition is met, the break statement will be executed, and it will cause the premature termination of a for loop.

Consider another example of a labeled for loop: 

outer:
for (int x = 0; x < 100; ++x)
{
      for (int y = 0; y < 100; ++y)
      {
         int   z = x*y;
         if (z == 6003)
            break outer;
         else if (z == 2475)
            break;

         } // end for

} // end for

The second break statement does not have a label. It will be executed only when the value of z becomes 2475, at which time only the innermost loop will terminate; it will not affect the outer loop.

The continue Statement

The continue statement is used inside a loop to skip the remaining statements in the loop and to go to the top of the loop. At the top of the loop, the Boolean expressions, if any, are evaluated and the rest of the loop execution continues.

The continue statement is used to skip the current iteration of the loop and go on to the next iteration. If a continue statement is executed in a while statement, the Boolean expression at the top of the loop is evaluated next. If a continue statement is executed in a do-while statement, the Boolean expression at the bottom of the loop is evaluated next. And if a continue statement is executed inside a for loop, the control is transferred to the increment expression (incrExpr).

The syntax of the continue statement is the reserved word continue on a line by itself with an optional label: 

continue [label];

When the label is used, the loop will continue to the loop that has the label. The inner loop will restart when it is executed again.

Consider the following example that uses a continue statement: 

for (int x = 1; x < 999; x++)
{
     y += x*3;
     if  (x == 333)
          continue;

     y += 5;

} // end for
