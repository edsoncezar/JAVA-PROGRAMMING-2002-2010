The Enterprise Development Challenge... Transactions - The ACID Properties

  • Atomicity

    • A transaction either commits or aborts

  • Consistency

    • A transaction is a correct transformation of system state

  • Isolation

    • Concurrent transactions are isolated from the updates of other incomplete transactions

  • Durability

    • Once a transaction commits, its effect will persist even if there are system failures
