The Enterprise Development Challenge... Naming Service - What is it?

  • Provides a natural, understandable way of identifying and associating names with data or objects

  • Provides the service of associating names with data or objects

  • Provides a Server layer and a Client layer

    • Server Layer - The Naming Service Server maintains & resolves actual name-to-object bindings and controls access to the directory service
    • Client Layer - Applications use the Naming Service Client to communicate with the directory service
