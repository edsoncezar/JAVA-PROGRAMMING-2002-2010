Estrutura do JMX

  • distributed layer:

    • responsável por tornar os agentes JMX disponíveis para o mundo
    • dois tipos de interação com esta camada:
      • Um tipo é através de adaptadores que dão visibilidade aos MBeans através de diferentes protocolos como HTTP e SNMP
      • O outro é através de conectores que espoem a API dos agentes para outras tecnologias distribuídas como Java RMI
