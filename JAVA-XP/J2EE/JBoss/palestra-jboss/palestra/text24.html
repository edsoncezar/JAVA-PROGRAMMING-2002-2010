Estrutura do JMX

  • agent layer:

    • Um agente não requer conhecimento sobre os recursos que ele gerencia ou sobre a aplicação gerenciadora que expõe os MBeans
    • agentes são implementados por desenvolvedores de sistemas de gerenciamento

  • instrumentation layer:

    • camada mais próxima dos recursos gerenciáveis
    • compreende os MBeans registrados pela agent layer
    • os MBeans são JavaBeans que possuem uma interface gerenciável. Se o recurso não conversa Java nativamente, o MBean atua como um tradutor do agente para o recurso
