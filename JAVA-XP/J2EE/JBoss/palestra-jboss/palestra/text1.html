O que é o JBoss?

  • Servidor de aplicações Java com suporte completo a especificação J2EE 1.3.

  • Desenvolvido sob a licença Open Source LGPL. Você não precisa comprar licenças para usá-lo!

  • Servidor de aplicações Java ganhador do prêmio JavaWorld 2002 como sendo o melhor de sua categoria (concorreu com iPlanet, WebLogic, Oracle9iAS, ...)

  • Servidor J2EE com maior número de downloads no mundo. Só em 2002 foram 2 milhões.

  • Servidor + utilizado no ambiente de desenvolvimento (segundo ToguetherSoft)
