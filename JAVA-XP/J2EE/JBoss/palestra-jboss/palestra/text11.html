Como instalar o JBoss?

  • Tenha o JDK versão 1.3 ou superior instalado

  • Tenha o ANT instalado (obrigatório somente no caso de ser feita a construção dos binários do JBoss ou para rodar exemplos)

  • Baixe os binários ou o código fonte de

    • http://www.jboss.org

  • A instalação a partir dos binários se resume a descompactação de um arquivo .zip

  • A instalação a partir do código fonte necessita da instalação do CVS (para obtenção das últimas versões) ou do download de um snapshot (arquivo compactado de uma versão ou dia)
