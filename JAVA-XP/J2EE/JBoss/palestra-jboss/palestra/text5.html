O que é Open Source?

  • Surgiu em busca de objetivos éticos, não técnicos

  • Preserva as liberdades dos usuários de computadores:

    • Liberdade de usar e distribuir o software
    • Liberdade de modificar o software e de distribuir as modificações

  • Também chamado de “Software Livre” ou “Software Aberto” quando se busca convencer as empresas a adota-lo devido apenas à sua superioridade técnica
