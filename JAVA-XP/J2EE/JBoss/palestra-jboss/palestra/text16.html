Como iniciar o JBoss?

  • Tenha já configurada, a variável de ambiente JAVA_HOME apontando para o diretório de instalação do java

  • Configure a variável de ambiente JBOSS_HOME para o diretório onde for realizada a instalação do JBoss

  • Inicie o JBoss invocando run.sh ou run.bat do diretório bin abaixo de JBOSS_HOME

  • Tente acessar o link abaixo averiguar a instalação:

    • http://localhost:8080/jmx-console
