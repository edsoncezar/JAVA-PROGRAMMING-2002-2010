Estrutura do JMX

  • Carregamento dinâmico

    • representado pelo serviço MLet (programa Java utilizado para instanciar MBeans obtidos de uma URL remota
    • o serviço MLet permite que se instancie e registre um servidor MBean um ou mais MBeans vindos da rede. Isto é feito a partir de um arquivo xml com as informações do MBean
    • o serviço MLet é implementado como um MBean e registrado no MBean servidor podendo ser utilizado tanto por aplicações de gerenciamento quanto por outros MBeans registrados no agente JMX
