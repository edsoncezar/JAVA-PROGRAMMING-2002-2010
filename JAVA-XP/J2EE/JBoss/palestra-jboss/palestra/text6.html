Porque usar o JBoss?

  • É de alta qualidade e gratuito.

  • É implementado totalmente em Java, o que garante portabilidade para diversos ambientes

  • É modular: é possível adicionar/remover novos serviços

  • O JBoss tem o código aberto. Quando programadores podem ler, redistribuir e modificar o código de um pedaço de software, o software evolui. As pessoas podem aprimorá-lo, e corrigir bugs. E isto ocorre em uma velocidade muito maior do que em um software proprietário.
