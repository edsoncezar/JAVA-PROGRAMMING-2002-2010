O que é o J2EE?

  • Sigla p/ Java 2 Enterprise Edition

  • É uma especificação de API's Java

  • É uma implementação de referência para estas API's

  • É um conjunto de aplicações de teste p/ verificar a aderência dos servidores a esta especificação

  • Inclui documentação para simplificar a construção de aplicações com requisitos altos de escalabilidade e disponibilidade (BluePrints)

  • É um local onde suas aplicações componentizadas com o Java, rodam com serviços oferecidos por containeres!
