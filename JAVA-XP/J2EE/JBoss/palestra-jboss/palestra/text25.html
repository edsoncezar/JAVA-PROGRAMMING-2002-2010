Estrutura do JMX

  • MBeans

    • classe Java não abstrata que contém:
      • um construtor público
      • a implementação da interface MBean correspondente ou a interface de um MBean dinâmico (DynamicMBean)
      • opcionalmente, a implementação da interface NotificationBroadcaster
    • são considerados MBeans padrões aqueles que implementam sua própria interface
    • MBeans dinâmicos implementam a interface DynamicMBean
    • Os agentes JMX manipulam MBeans de ambos os tipos de forma trasparente
