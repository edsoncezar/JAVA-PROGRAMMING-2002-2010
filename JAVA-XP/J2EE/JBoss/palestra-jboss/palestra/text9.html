Ferramentas Open Source auxiliares

  • ANT

    • Possibilita a construção de aplicativos sem a utilização de IDE's de forma semelhante ao utilitário make

  • XDoclet

    • Automatiza a geração de arquivos java e de configuração

  • CVS

    • Controla versões de arquivos fonte, automatiza o processo de criação de releases

  • JUnit

    • Framework automatizador de testes. Padrão para quem trabalha com eXtreme Programming
