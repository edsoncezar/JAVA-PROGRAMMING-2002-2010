Estrutura do JMX

  • agent layer:

    • principal componente: MBean server que é um objeto Java atuando como um registro para outros MBeans
    • provê quatro serviços que tornam o gerenciamento dos MBeans simples: tempo, monitoração, carregamento dinâmico de MBeans e serviços de relacionamento
    • provê acesso aos recursos gerenciáveis para a aplicação de gerenciamento
    • Um agente pode ser executado na mesma JVM que contém os recursos gerenciáveis (MBeans) ou ser remoto
