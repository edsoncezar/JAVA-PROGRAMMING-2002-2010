Estrutura de diretórios do JBoss

  • bin

    • Jars e scripts para se iniciar o JBoss

  • client

    • Jars necessários para a execução de clientes

  • server

    • Arquivos de conjuntos de configuração do JBoss server. A configuração default está no diretório server/default. O JBoss vem com três conjuntos de configuração: default, minimal e all

  • lib

    • Jars de startup utilizados pelo JBoss
