Estrutura do JMX

  • MBean padrão:

    • os recursos a serem gerenciados são acessados diretamente através de seus atributos e operações
    • os agentes utilizam a característica de introspecção do Java para obterem informações do MBean

  • MBean dinâmico:

    • os recursos a serem gerenciados são acessados indiretamente através de chamadas de métodos
    • ao invés de introspecção, os agentes devem chamar um método que encontre o nome e a natureza dos atributos e das operações do MBean
