Clustering no JBoss

  • As seguintes características estão disponíveis através do JBoss Clustering

    • Os nós do cluster se encontram sem nenhuma configuração adiocional
    • A árvore JNDI é replicada entre todos os membros do Cluster
    • Tolerância a falhas e balanceamento de carga para JNDI, RMI e EJBs
    • Replicação de estado para componentes EJB stateful
    • Deployment distribuído para EJBs e serviços JBosss
