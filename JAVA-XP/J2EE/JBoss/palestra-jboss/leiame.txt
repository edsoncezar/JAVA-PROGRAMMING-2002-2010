    Neste diretório encontra-se a palestra ministrada por mim no dia 23/01/2002 para o grupo de usuário Java do DF (DFJUG).
    Para que todos os exemplos possam ser compilados corretamente você deverá ter instalado o ANT, o JBoss, o XDoclet e configurado as seguintes variáveis de ambiente:

        JAVA_HOME -> deve apontar para o diretório de sua instalação do JDK
        ANT_HOME -> deve apontar para o diretório de instalação do ANT
        JBOSS_HOME -> deve apontar para o diretório de sua instalação do JBoss
        XDOCLET_HOME -> deve apontar para o diretório de instalação do XDoclet

    Exemplo de configuração feita no meu Windows 2000:

        set JAVA_LIBDIR=C:\devtools\java
        set JAVA_HOME=%JAVA_LIBDIR%\jdk1.3.1_06
        set JBOSS_HOME=%JAVA_LIBDIR%\jboss-3.0.4_tomcat-4.1.12
        set ANT_HOME=%JAVA_LIBDIR%\jakarta-ant-1.5.1
        set XDOCLET_HOME=%JAVA_LIBDIR%\xdoclet-1.1.2

    A documentação para os exemplos testJBoss e testJBossCMP2 pode ser encontrada no livro JBoss Getting Started (disponível para download no site do JBoss).
