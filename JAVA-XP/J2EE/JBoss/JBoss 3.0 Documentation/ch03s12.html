Examples for specific databases

Please contribute your working example now!

Examples for CURRENT configurations of JBoss 3.0RC1 and later are maintained in cvs at http://cvs.sourceforge.net/cgi-bin/viewcvs.cgi/jboss/jbosscx/src/etc/example-config/ or in connector/src/etc/example-config of a cvs checkout. You should be able to find a version apropriate for your jboss by matching the cvs tags to that for your jboss version.

For previous, obsolete, versions of jboss jca support,example ConnectionFactoryLoader configuration files can be seen in the cvs attic at http://cvs.sourceforge.net/cgi-bin/viewcvs.cgi/jboss/manual/src/examples/resources/jca/. For these obsolete configurations, please examine the checkin comments to find the version appropriate for the version of jboss 3 that you are using.

None of these examples, new or old, will work with any jboss 2.x version.
