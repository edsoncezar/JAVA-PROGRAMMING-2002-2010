- Assuntos tratados:
    - Geração do arquivo web.xml;
    - Geração de descritores de bibliotecas de tags;
    - Geração de lista de todo's;

- Ferramentas utilizadas:
    - ANT (1.5.4)
    - XDoclet (1.2b3)
    - JBoss (3.2.1)

- Como executar o exemplo:
    - Execute o comando:
    c:\mini-curso-xdoclet\src\02-webdoclet\ant deploy

    - Inicie o JBoss:
    c:\mini-curso-xdoclet\src\02-webdoclet\start run

    - Inicei o browser e acesse a url:
    c:\mini-curso-xdoclet\src\02-webdoclet\start http://localhost:8080/webdoclet
