- Assuntos tratados:
    - Eterprise JavaBeans
    - CMP 2.0 (Container Managed Persistence)
    - Geração automática de EJB's através do Middlegen e do XDoclet

- Ferramentas utilizadas:
    - ANT (1.5.4)
    - Middlegen (2.0-b1)
    - XDoclet (1.2b3)
    - HSQLDB (1.7.2)
    - JUnit (3.8.1)
    - Velocity (1.3.1) - Necessário p/ execução do Middlegen
    - JBoss (3.2.1)

- Procedimentos (Execute-os passo a passo):
    - inicie o JBoss;
    (Isto é obrigatório, pois o banco será criado através do serviço HSQLDB iniciado pelo JBoss)

    - ant force-middlegen 
    (Isto criará os entity beans necessários para a execução da aplicação - clique no botão "Generate" para criar os entity beans! Este passo é opcional - só executar se os entity beans não existirem (caso inicial) ou a estrutura das tabelas mudar)

    - ant deploy
    (Isto irá implantar a aplicação no JBoss)

    - ant test
    (Isto executará o JUnit que invocará os métodos do session bean que acessa os entity beans da aplicação)
