Brasília-DF, 06/Out/2003.

Olá,

    Este mini-curso apresenta algumas das funcionalidades do XDoclet (http://xdoclet.sourceforge.net). São demonstrados exemplos que envolvem a utilização desta ferramenta para se criar templates, gerar os arquivos web.xml, *.hbm.xml (Hibernate) e os deployments descriptors necessários para a implantação de EJB's no JBoss (ejb-jar.xml, jboss-*.xml).
    É necessário para a execução das aplicações, a instalação do ANT e do JBoss (http://www.jboss.org). As demonstrações foram feitas utilizando-se as seguintes versões: apache-ant-1.5.4 e jboss-3.2.1_tomcat-4.1.24. As outras bibliotecas necessárias para a compilação/execução das aplicações (commons-collections, hibernate, hsqldb, jakarta-log4j, junit, middlegen, velocity e xdoclet) estão disponíveis (somente os jars necessários) dentro do diretório lib. Para cada aplicação disponível dentro do diretório src (01-... até 04-...) há um arquivo leiame.txt com as intruções necessárias.
    Alguns exemplos utilizam a variável de ambiente JBOSS_HOME, que deve apontar para o local onde foi realizada a instalação do JBoss. Crie esta variável antes de executar os exemplos. Coloque também no PATH, o diretório bin do JBOSS_HOME e do ANT_HOME (local onde foi realizada a instalação do ANT), para que possam ser invocados os comandos run.bat e shutdown.bat (JBoss) além do ant.bat (ANT).
    Há dentro do diretório slides, links (Internet Explorer) para slides do curso J820 (http://www.argonavis.com.br/cursos/java/j820/index.html), que a Argo Navis ministra e que falam sobre o ANT e sobre o XDoclet.
    Dúvidas com relacão a compilação e/ou execução dos programas fornecidos podem ser repassadas diretamente a mim, através do meu e-mail: pj@paulojeronimo.eti.br.
