JBuilder 6

JBuilder 6 is the most comprehensive set of award-winning, visual development tools for creating Pure Java, platform-independent business applications, applets, servlets, and JavaBeans for the Java 2 platform.

System Requirements

  • 128 MB (recommended minimum)
  • 256 MB (recommended)
  • 300 MB hard disk space (depending on features installed)
  • Microsoft® Windows 2000 (SP2), XP, or NT 4.0 (SP6)
  • Intel® Pentium® II 233 MHz or compatible

Please read these instructions entirely before installing JBuilder 6.

  1. Install JBuilder 6 Personal.
  2. Install JBuilder 6 Documentation. (optional)
  3. Install JBuilder 6 Additional Samples. (optional)
  4. Run JBuilder 6.

For additional information on:

  • Configuring and using JBuilder 6, please see the Release Notes.
  • New features available in this version of JBuilder 6, see What's New in JBuilder 6.

Installing JBuilder 6

JBuilder 6 contains a version of JDK 1.3.1 (which includes JPDA). You should use this version instead of any earlier version that you might have.

To install JBuilder 6:

  1. Unzip the jb6_windows.zip file you downloaded.
  2. Double-click on per_install.exe to start the installation program.
  3. If needed, change the default language for JBuilder 6, then click OK.
  4. Read the information provided, then click Next.
  5. Accept the license agreement when prompted, then click Next.
  6. Choose the directory where you want JBuilder 6 installed, then click Next. Do not choose a directory that contains spaces in its name. JBuilder will not run if the name of the directory in which it is installed contains spaces.
  7. Review the installation options that you have selected. If the options are correct, then click Install to begin the installation.
  8. When the installation program is completed, click Done.

Install JBuilder 6 Documentation

To install the JBuilder 6 Documentation, you should also have at least 80 MB of available hard disk space.

To install the JBuilder 6 Documentation:

  1. If you have not done so already, download the JBuilder 6 Documentation from JBuilder Personal Additional Software.
  2. Unzip the jb6docs.zip file you downloaded.
  3. To launch the installation program, do one of the following:
    • Double-click on doc_install.exe.
    • From the command-line, enter doc_install.exe.
  4. If needed, change the default language for the JBuilder 6 Documentation installation, then click OK.
  5. Follow the prompts and click Next.
  6. Accept the license agreement when prompted, then click Next.
  7. Choose the directory where JBuilder 6 was installed, then click Next.
  8. When the installation program is completed, click Done.

Note: The documentation is also available in PDF format from the JBuilder Technical Publication web page.

Install JBuilder 6 Additional Samples

To install the JBuilder 6 Additional Samples, you should also have at least 8 MB of available hard disk space.

To install the JBuilder 6 Additional Samples:

  1. If you have not done so already, download the JBuilder 6 Additional Samples from JBuilder Personal Additional Software.
  2. Unzip the jb6samples.zip file you downloaded.
  3. To launch the installation program, do one of the following:
    • Double-click on smp_install.exe.
    • From the command-line, enter smp_install.exe.
  4. If needed, change the default language for the JBuilder 6 Additional Samples installation, then click OK.
  5. Follow the prompts and click Next.
  6. Accept the license agreement when prompted, then click Next.
  7. Choose the directory where JBuilder 6 was installed, then click Next.
  8. When the installation program is completed, click Done.

Uninstalling JBuilder 6

To uninstall JBuilder 6, do one of the following:

  • Run the following from the directory where you installed JBuilder 6:
    • UninstallPersonal\UninstallPersonal.exe
    • UninstallPersonal\UninstallDocumentation.exe
    • UninstallPersonal\UninstallSamples.exe
  • Choose Start|Programs|JBuilder 6|Uninstall JBuilder 6 Personal.
  • Choose Settings|Control Panel|Add/Remove programs and select JBuilder 6 Personal from the list.
